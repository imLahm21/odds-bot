# 中超基本面权重修正

> **触发条件**：中超（及状态波动类似的次级联赛）赛事必读。
> 底层逻辑：中超赛程密集（联赛+足协杯+亚冠）+ 体能管理差 → 状态波动大；主场胜率基准仅 38%~42%（五大联赛 45%~50%），主场优势与排名差的参考价值均低于欧洲联赛。

## A. 近况崩溃期的主场反弹规律

> 来源：浙江 2-1 深圳新鹏城（2026-05-02 中超）。两队近况均差误判"平局概率最高"，实际主队反弹取胜。

| 条件 | 判断 |
|------|------|
| 主队连续 3 场+ 不胜 + 客队客场连续 2 场+ 不胜 | **主场方反弹概率 > 客场方** |
| 主队连续 3 场+ 不胜 + 客队客场状态正常 | 需综合判断 |
| 两队近况均崩溃 | 主场方反弹概率更高（主场球迷+排名压力+训练环境） |

低迷队主场承受更大反弹压力，客场低迷队缺乏反弹动力和环境。低迷期历史交锋权重应**上调**（详见 [feedback_h2h_weight.md](feedback_h2h_weight.md) C 节）；近况必须按对手质量量化（[feedback_home_away_quality.md](feedback_home_away_quality.md) C 节）。

## B. 中超主场反弹潜力四条件

> 来源：浙江 4-1 山东泰山（2026-05-20 中超）。过度放大客队状态优势，低估主队主场交锋能力。存档：[20260520_case_01_zhejiang_vs_shandong.md](20260520_case_01_zhejiang_vs_shandong.md)。

四项同时满足 → 主场反弹潜力作为核心权重：

| 条件 | 阈值 |
|------|------|
| ① 主队历史主场交锋不败率 | ≥ 60%（近 5 场） |
| ② 客队近 10 场客场大败前科 | 存在净负 ≥ 3 球记录 |
| ③ 主队近况输球对手质量高 | 输给联赛前 3 强队 |
| ④ 积分差 | ≤ 10 分 |

**权重修正**：客队近期状态优势 25%→**10%**；主队近况低迷 20%→**10%**；主队主场交锋能力 15%→**35%**；中超主场反弹规律 10%→**25%**；盘口信号 30%→20%。

**置信度上限 80**：盘口信号与反弹潜力一致 → 75~80；冲突 → 优先基本面，65~75。
**边界**：本赛季积分差 ≤ 3 分时**不触发** H2H 35% 上调，反而下调至 20%（见 [feedback_h2h_weight.md](feedback_h2h_weight.md) B 节）。

## C. 中超单场大败 ≠ 持续崩溃

> 来源：河南 0-2 浙江（2026-05-30 中超）。浙江 0-5 惨败 6 天后被判"真实崩溃"，实际客场 0-2 完胜。

判定崩溃前逐项验证：

| 验证项 | 非崩溃（权重降至 10~15%） | 真实崩溃（权重 30~35%） |
|--------|------------------------|----------------------|
| 大败后是否有大胜记录 | 有（0-5 后 4-1 胜强队）= 波动大 | 无，连续 3 场大败 |
| 大败对手是否强队 | 是（前 3 名）= 输强队 | 否（中下游）= 真实低迷 |
| 近 3 场是否连败 | 否（胜负交替）= 波动 | 是 = 真实崩溃 |
| 大败场地是否与本场相反 | 是（客场大败+本场主场）= 主客差异 | 否 |

大败后立即大胜强队 = 攻击力爆发 + 防守不稳，非单向崩溃；单场大败权重降至 15%，近 3 场整体趋势权重上调至 25%。

## D. 「排名差 + 主场不败」组合的权重上限与进攻低迷修正

> 来源：重庆铜梁龙 2-3 北京国安（2026-05-30 中超）。积分差 12 分 + 主场不败被判"碾压"权重 35%，忽视近 4 场仅 1 球的进攻低迷。存档：[20260531_case_03_chongqing_vs_beijing.md](20260531_case_03_chongqing_vs_beijing.md)。

**规则 D1：中超「排名差 ≥10 分 + 主场近 5 场不败」组合权重上限 25%**（五大联赛 35%），且必须验证近期进球数 + 不败质量。

**规则 D2：主场不败质量分级**：

| 主场不败质量 | 判定 | 权重 |
|------------|------|------|
| 含 2 球+ 净胜（2-0、3-1） | 高质量 | 维持 25% |
| 多为 1 球净胜（1-0、2-1） | 中等 | 下调至 20% |
| **多为 0-0、1-1 小比分** | **低质量（防守稳但进攻存疑）** | **下调至 15%** |

**规则 D3：近期进球数修正**（进球数比胜负数更反映当前进攻能力）：

| 主队近 4 场 | 客队近 3 场 | 权重调整 |
|-----------|-----------|---------|
| 主队 ≥6 球 + 客队 ≤3 球 | 进攻方主队 | +5% |
| 均 3~5 球 | 均衡 | 无调整 |
| **主队 ≤2 球 + 客队 ≥6 球** | 进攻方客队 | **-10%** |
| 主队 ≤2 球 + 客队 ≤2 球 | 双方低迷 | -5%（平局概率上升） |

低质量不败 + 进攻低迷叠加时基本面权重可压至 10% 左右；与盘口/凯利信号冲突时以凯利信号为主。

## E. 保级死战动力 > 双重极端信号（极端动力差异场景）

> 来源：天津津门虎 1-0 大连英博（2026-05-31 中超）。机械套用凯利双重极端强制反转推客队，实际保级方主胜。存档：[20260531_case_04_tianjin_vs_dalian.md](20260531_case_04_tianjin_vs_dalian.md)。

**规则 E1：满足以下 ≥ 3 项 = 极端动力差异场景，关闭凯利双重极端强制反转**（[feedback_kelly_signals.md](feedback_kelly_signals.md) F 节）：

| 条件 | 阈值 |
|------|------|
| ① 主队保级死战 | 排名 ≥ 15 且距安全区 ≤ 6 分 |
| ② 客队争冠/保级无望 | 距榜首 ≥ 12 分 或 距降级区 ≥ 10 分 |
| ③ 客队客场胜率低 | < 50% |
| ④ 剩余轮次少 | ≤ 8 轮 |

触发后权重：保级死战动力 **55%**、客队动力不足 20%、盘口信号 **25%**（下调）。保级死战 = 生死存亡级动力，无法用凯利量化，远超常规主场优势（15~25%）与争冠关键战（30~40%）。

**规则 E2：客队「无欲无求」识别**：争冠无望/保级无忧 + 客场胜率 < 50% + 有输弱队前科 → 客场易轮换保守，动力缺失权重 20%。

**规则 E3：保级死战场景下临场降盘 + 主水反升 = 风控平衡（非诱上）**：散户涌入主队、庄家抬主水平衡赔付。

**规则 E4：即时节点（开球后）数据不可用于赛前预测，但可验证逻辑**：即时方向与临场④完全相反 → 临场④信号可能是诱导，基本面逻辑优先。

**适用/不适用速查**：常规比赛/动力均衡 → 双重极端强制反转适用；**保级死战 vs 无欲无求 / 争冠决战 vs 保级队 / 德比 / 杯赛决赛（极端战意差）→ 不适用**。

## F. 中超通用修正（所有中超场次）

- **主场优势权重下调**：中超主场胜率 38%~42%，排名差 ≥ 5 名仅作参考信号（五大联赛为强信号）
- **「主场+排名差」组合置信度上限 75**（不可给 80+）
- **主场近期防守崩盘前科**（0-3、0-4 大败）→ 显著扣减置信度 10 分
- **主队近期主场小负前 3 强队**（1-2/0-1/0-2）或主场让球盘连续 2 场+ 未兑现 → 半球及以上让球盘额外扣减置信度 5 分（中超让球能力波动大）
- 客场连败队的反弹潜力须纳入考虑（不可只看绝对客场战绩）

## 本主题检查项

- [ ] 近况崩溃的主队对阵客场低迷队？→ 考虑主场反弹概率
- [ ] 主场反弹四条件（主场交锋不败率 ≥60% / 客队客场净负 ≥3 球前科 / 输球对手为强队 / 积分差 ≤10 分）是否逐项核对？
- [ ] 单场大败是否误判为持续崩溃？（大败后大胜 / 输强队 / 非连败 / 场地相反 → 权重 15%）
- [ ] 「排名差+主场不败」权重是否封顶 25%？不败多为 0-0/1-1 → 降至 15%；主队 ≤2 球+客队 ≥6 球 → 再扣 10%
- [ ] 极端动力差异（保级死战 vs 无欲无求，≥3 项）？→ 关闭凯利双重极端强制反转，动力权重 55%
- [ ] 保级死战下临场降盘+主水反升是否识别为风控平衡（非诱上）？
- [ ] 中超「主场+排名差」置信度是否 ≤ 75？主场防守崩盘前科是否扣 10 分？主场小负强队是否扣 5 分？
