# 主客场状态按对手质量量化

> **触发条件**：用户提供两队近 10 场数据时必读（步骤 1b 近况分析配套）。
> 核心原则：**不可简单统计「近 N 场 X 胜」作为状态指标**，必须区分「实力输球」（对手太强）和「状态输球」（对同级别也输）。

## A. 主场输强队 ≠ 主场能力崩溃

> 来源：Chelsea 2-1 Tottenham（2026-05-20 英超）。主场近 3 场全败（0-3 曼城、0-1 曼联、0-3 诺丁汉）被判"主场崩溃"权重 40%，实际 3 负均输给强队/状态火热队，主场对热刺 H2H 100% 不败兑现。

| 主场输球对手 | 信号意义 | "主场崩溃"权重 |
|------------|---------|--------------|
| 主要输前 3 强队/榜首集团（≥60%） | 输强队正常 | **15%** |
| 主要输同级别中游（≥60%） | 真实低迷 | 30% |
| 主要输弱队（≥40%） | 严重崩盘 | 40% |

## B. 客场「优于主场」须验证稳定性（A 的镜像）

> 来源：同上。Tottenham 客场 2 胜 1 负被判"客场战斗力强"权重 25%，实际胜负交替 + 小比分。

| 验证项 | 真实客场强 | 虚假信号 |
|--------|----------|---------|
| 对手质量 | 客场赢同级别或强队 | 赢弱队、输强队 |
| 稳定性 | 客场连续 3 场+ 不败 | 胜负交替，不稳定 |
| 净胜球 | 客场平均净胜球 > 0 | 接近 0（小胜小负） |

权重：连续不败 + 赢同级 + 净胜 > 0 → 25%；胜负交替 + 小比分 + 净胜 ≈ 0 → 10%；主要赢弱队 → 5%。

## C. 近况对手质量量化（通用规则，主客队均适用）

> 来源：浙江 2-1 深圳、上海海港 2-2 浙江、浙江 4-1 山东多案汇总。

| 近况输球对手等级 | 信号意义 |
|------------|--------------|
| 联赛前 3~4 强队（榜首集团） | **输给强队 ≠ 状态崩溃**（不可作为支持对方盘口方向的信号） |
| 同级别中游队 | 输给同级别 = 状态真实低迷 |
| 联赛后 4 弱队 | 输给弱队 = 严重崩盘（强信号） |

应用要点：

- 连续输给强队后遇弱队/中游队 → 反弹概率上升
- 客队「客场极弱」判断前先分级：3 负中输榜首集团占主导 → 判断不成立（海港案：浙江客场 3 负中 0-4 成都(第1)、0-3 大连(第3)）
- 客队近 10 场客场**净负 ≥ 3 球大败前科**（如 1-4、2-4）→ 中超主场反弹潜力条件之一（[feedback_csl_fundamentals.md](feedback_csl_fundamentals.md) B 节）
- 中超单场大败的四项验证 → [feedback_csl_fundamentals.md](feedback_csl_fundamentals.md) C 节

## 本主题检查项

- [ ] 主队近况输球是否主要输给前 3 强队？→ "主场崩溃"权重降至 15%
- [ ] 客队"客场优于主场"时是否验证三项（对手质量/稳定性/净胜球）？
- [ ] 主客队近况均已按对手质量分级（非简单数胜负）？
- [ ] 输给榜首集团占主导的"客场极弱/状态崩溃"判断是否已否决？
- [ ] 连续输强队后本场遇弱队/中游？→ 反弹概率上调
