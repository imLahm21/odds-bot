# 临场④超大异动 = 反向定价

## 核心规则

**临场④超大异动（盘口 ≥ 0.5 球 + 欧赔 ≥ 0.25）= 反向定价，非小幅修正**

当以下条件同时满足时，必须判定为庄家**反向定价**（不看好原方向）：
- 前 8 节点盘口相对稳定（变化 ≤ 0.25 球）
- 临场④（-0.5h）盘口变化 ≥ 0.5 球
- 临场④欧赔主胜变化 ≥ 0.25
- 主水方向与盘口方向一致（降盘 + 主水降，或升盘 + 主水升）

此时**不可套用**「欧亚同步降盘 = 定价修正 → 上盘小胜」（`dynamic 2.7`），因为异动幅度已超出"修正"范畴。

## Why

**临场④是知情资金窗口**：
- 临场④（-0.5h）是最接近开球的节点，庄家此时掌握最完整的临场信息（阵容、伤停、天气、场地、内幕消息）
- 超大异动（盘口 0.5 球）= 庄家发现此前 8 节点的定价**严重错误**，必须急剧修正
- 若仅是"小胜"修正，降盘幅度应为 0.25 球（如 -0.75→-0.5），而非 0.5 球（-0.75→-0.25）

**欧赔主胜大幅上升 = 不看好主队赢球**：
- 欧赔主胜 +0.25+ = 庄家大幅提高主胜赔率，**降低主胜赔付压力**
- 这不是"小胜"修正（小胜修正应为 +0.05~0.15），而是**不看好主队赢球**

## How to apply

### 阈值定义

| 异动幅度 | 盘口变化 | 欧赔主胜变化 | 判定 |
|---------|---------|-------------|------|
| **超大异动** | **≥ 0.5 球** | **≥ 0.25** | **反向定价** → 反向投注 |
| 中等异动 | 0.25~0.5 球 | 0.10~0.25 | 需结合凯利/水位判断 |
| 小幅修正 | < 0.25 球 | < 0.10 | 维持原判断 |

### 执行步骤

1. **检查前 8 节点盘口稳定性**：变化 ≤ 0.25 球 → 进入步骤 2
2. **计算临场④异动幅度**：
   - 盘口变化 = |临场④让球 - 临场③让球|
   - 欧赔变化 = |临场④主胜 - 临场③主胜|
3. **判定异动级别**：
   - 盘口 ≥ 0.5 球 + 欧赔 ≥ 0.25 → **超大异动 = 反向定价**
   - 盘口 0.25~0.5 球 + 欧赔 0.10~0.25 → 中等异动（需结合凯利）
4. **确认方向**：
   - 降盘 + 欧主胜上升 → 不看好主队 → **下盘（客队）**
   - 升盘 + 欧主胜下降 → 不看好客队 → **上盘（主队）**

### 置信度调整

- 超大异动 + 凯利反转（见 [[feedback_kelly_reversal_priority]]）→ 置信度 +40 分
- 超大异动 + 凯利维持原方向 → 置信度 +25 分（存在冲突，降低权重）
- 超大异动 + 基本面支持反向 → 置信度 +15 分

## 相关记忆

- [[feedback_kelly_reversal_priority]]（凯利反转优先）— 本规则的验证层：超大异动通常伴随凯利反转
- [[feedback_euroasian_sync_upgrade]]（欧亚同步升盘=真实定价）— 本规则的镜像：升盘场景的真实定价
- [[feedback_lure_down_variant]]（临场诱下组合）— 本规则的边界：降盘 + 主水先降后反升 = 诱下，非反向定价

## 案例

**河南 0-2 浙江（2026-05-30 中超）**：
- 前 8 节点盘口锁定 -0.75
- 临场④盘口 -0.75→-0.25（降 0.5 球）+ 欧主胜 1.80→2.09（+0.29）+ 主水降 2.03→1.84
- 误判为"欧亚同步降盘 = 定价修正 → 河南小胜"
- 实际：浙江 0-2 客胜，临场④超大异动 = 反向定价，庄家不看好河南

## 边界条件

**不触发本规则的场景**：
1. 前 8 节点盘口波动大（如 -1.0→-0.75→-0.5）→ 持续修正，非超大异动
2. 临场④主水反升 ≥ 10 点 → 触发 [[feedback_lure_down_variant]]（诱下组合），非反向定价
3. 临场④欧赔主胜下降 → 升盘场景，走 [[feedback_euroasian_sync_upgrade]]（真实定价）
