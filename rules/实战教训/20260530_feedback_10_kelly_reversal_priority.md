# 凯利反转优先于凯利绝对值

## 核心规则

**凯利方向比绝对值更重要**

当以下条件满足时，必须判定为**凯利体制反转**：
- 前期凯利持续某方向（如主凯利 < 0.90 持续 3+ 节点）
- 临场④凯利反向变化 ≥ 0.05（如主凯利回升 +0.05+）
- 对方凯利同步反向变化 ≥ 0.05（如客凯利下降 -0.05+）

此时**不可关注凯利绝对值**（如"0.926 仍 < 0.96 = 信号维持"），因为**趋势反转已形成**。

## Why

**凯利指数是庄家对打出概率的动态定价**：
- 凯利 < 1.0 = 庄家压制该方向赔率，看好该方向打出
- 凯利 > 1.0 = 庄家抬高该方向赔率，不看好该方向打出
- **凯利变化方向 = 庄家态度变化**，比绝对值更能反映庄家最新判断

**临场④是最后修正窗口**：
- 临场④（-0.5h）是最接近开球的节点，庄家此时掌握最完整信息
- 凯利反转 = 庄家修正此前判断，**最新态度 > 历史态度**

## How to apply

### 阈值定义

| 凯利变化幅度 | 判定 |
|------------|------|
| **≥ 0.05** | **反转信号** → 庄家态度反转 |
| 0.03~0.05 | 灰色地带 → 需结合盘口/欧赔判断 |
| < 0.03 | 微调 → 维持原判断 |

### 执行步骤

1. **计算凯利变化**：
   - 主凯利变化 = 临场④主凯利 - 前期最低主凯利
   - 客凯利变化 = 临场④客凯利 - 前期最高客凯利

2. **判定反转**：
   - 主凯利变化 ≥ +0.05 + 客凯利变化 ≤ -0.05 → **凯利体制反转 → 看好客队**
   - 主凯利变化 ≤ -0.05 + 客凯利变化 ≥ +0.05 → **凯利体制反转 → 看好主队**

3. **忽略绝对值**：
   - 即使临场④主凯利仍 < 0.96，只要变化 ≥ +0.05，仍判定为反转
   - 即使临场④客凯利仍 > 1.03，只要变化 ≤ -0.05，仍判定为反转

### 置信度调整

- 凯利反转 + 临场④超大异动（见 [[feedback_late_stage_massive_shift]]）→ 置信度 +40 分
- 凯利反转 + 盘口/欧赔同向 → 置信度 +30 分
- 凯利反转 + 盘口/欧赔反向 → 置信度 +15 分（存在冲突，降低权重）

## 案例

**河南 0-2 浙江（2026-05-30 中超）**：
- 中盘②~临场③ 主凯利 0.853~0.887（持续 5 节点 < 0.90）
- 临场④主凯利 0.926（+0.073）+ 客凯利 1.040（-0.061）
- 误判为"0.926 仍 < 0.96 = 信号维持（强度降为中强）"
- 实际：凯利反转 = 庄家转向看好浙江，浙江 0-2 客胜

## 边界条件

**不触发本规则的场景**：
1. 凯利变化 < 0.03 → 微调，维持原判断
2. 仅主凯利反转，客凯利不变 → 单边反转，需结合盘口/欧赔判断
3. 升盘前凯利极低（< 0.90）+ 升盘后回升 → 触发 [[feedback_preupgrade_kelly_priority]]（升盘前凯利优先），非反转
