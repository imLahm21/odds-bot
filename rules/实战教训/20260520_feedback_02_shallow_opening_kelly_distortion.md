# 浅盘凯利信号失真修正

## 规则来源

浙江 4-1 山东泰山（2026-05-20 中超）— 初盘① Pinnacle 客胜凯利 1.042，误判为"升盘前真实信号"，实际主队大胜

## 核心规则

当初盘开小（实际盘口 < 理论盘口 0.25 球+）时，客胜凯利 > 1.0 可能是**盘口浅开导致的数学结果**，而非庄家真实看好客队。

### 失真机制

| 盘口性质 | 客胜赔率特征 | 客胜凯利表现 | 失真原因 |
|---------|------------|-------------|---------|
| 初盘合适/开大 | 正常定价 | < 1.0 = 看好客队 | 无失真 |
| **初盘开小** | **结构性偏高** | **> 1.0 可能是数学结果** | **浅开 → 客队让球负担轻 → 客胜赔率相对市场偏高** |

## Why（为什么）

- **浅开的数学效应**：初盘开小意味着主队让球深度低于理论值，客队让球负担轻，客胜赔率相对市场偏高
- **凯利计算的结构性偏差**：凯利 = 该公司赔率 / 市场平均赔率。当某公司初盘浅开，其客胜赔率高于市场平均，导致客胜凯利 > 1.0
- **这不是庄家态度**：客胜凯利 > 1.0 是盘口结构导致的数学结果，不等于庄家真实看好客队

## How to apply（如何应用）

### 凯利强信号阈值的浅盘修正

| 初盘性质 | 客胜凯利强信号阈值 | 灰色地带 | 说明 |
|---------|------------------|---------|------|
| 初盘合适/开大 | < 0.96（看好）、> 1.05（不看好） | 0.96~1.05 | 标准阈值 |
| **初盘开小** | **< 0.95（看好）、> 1.08（不看好）** | **0.95~1.08** | **阈值放宽，灰色地带扩大** |

### 灰色地带凯利的正确处理

当客胜凯利落入灰色地带时，**不可单独作为定性依据**，必须结合以下信号综合判断：

| 验证项 | 真实看好客队 | 凯利信号失效 |
|--------|------------|-------------|
| 升盘后主水方向 | 下降或维持（≤ 5 点） | **反升 ≥ 15 点** |
| 即时欧赔延续性 | 延续下降或稳定 | **回调 ≥ 0.05** |
| 升盘后主凯利 | < 1.05 | **> 1.08** |

### 判定流程

```
初盘开小？
  ├─ 否 → 按标准凯利阈值处理（< 0.96 看好客队，> 1.05 不看好客队）
  └─ 是 → 客胜凯利在 0.95~1.08 区间？
         ├─ 否 → 按浅盘修正阈值处理（< 0.95 看好客队，> 1.08 不看好客队）
         └─ 是 → 灰色地带，不可单独定性
                 └─ 验证升盘后主水方向 + 即时欧赔延续性 + 主凯利
                    ├─ 主水反升 ≥ 15 点 + 欧赔回调 ≥ 0.05 + 主凯利 > 1.08 → 凯利信号失效，反向看好主队
                    └─ 主水下降或维持 + 欧赔延续下降 + 主凯利 < 1.05 → 真实看好客队
```

## 与既有规则的关系

### 与"升盘前凯利趋势优先"的边界条件

[[feedback_preupgrade_kelly_priority]] 规则有边界条件：

| 场景 | 升盘前凯利趋势优先 | 需重新验证 |
|------|------------------|-----------|
| 初盘合适/开大 + 升盘前凯利 < 0.97 | **适用**（升盘前真实信号） | — |
| **初盘开小 + 升盘前凯利 1.03~1.05** | — | **不适用，必须验证升盘后水位/欧赔/主凯利** |

### 与"客胜凯利 < 0.96 优先级反转"的关系

[[feedback_kelly_priority_inversion]] 规则的阈值在浅盘场景下需修正：
- 标准场景：Pinnacle 客胜凯利 < 0.96 = 强信号
- **浅盘场景**：Pinnacle 客胜凯利 < **0.95** = 强信号（阈值下调 0.01）

## 案例数据

### 浙江 4-1 山东泰山（2026-05-20 中超）

| 项目 | 数据 |
|------|------|
| 理论盘口 | 0.5（依据欧赔 3.07/3.60/2.12） |
| 实际初盘 | 0.25 |
| 判定 | **初盘开小**（开小 0.25） |
| 初盘① Pinnacle 客胜凯利 | 1.042 |
| 凯利区间判定 | 落入浅盘灰色地带（0.95~1.08） |
| 升盘后主水变化 | 1.83→2.01→2.08（+25 点） |
| 即时欧赔回调 | 2.65→2.75（+0.10） |
| 升盘后主凯利 | 1.083→1.127→1.085→1.118 |
| **最终判定** | **客胜凯利 1.042 信号失效，反向看好主队** |

**实际结果**：浙江 4:1 主场大胜，客胜凯利信号完全失效

## 防错要点

1. **初盘开小是前置条件**：只有初盘开小场景才触发凯利失真修正，标准盘口或深盘按标准阈值处理
2. **灰色地带不可单独定性**：客胜凯利 1.03~1.05 在浅盘场景下必须结合升盘后水位/欧赔/主凯利综合判断
3. **不可机械套用"升盘前凯利趋势优先"**：该规则在浅盘 + 灰色地带凯利场景下不适用
4. **阈值修正幅度有限**：浅盘修正仅将强信号阈值调整 0.01~0.03，不可过度放宽
