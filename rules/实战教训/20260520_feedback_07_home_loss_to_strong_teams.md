# 主场输强队 ≠ 主场能力崩溃

## 核心规则

**主场近期输球的对手质量量化**

不可简单统计"主场近 N 场战绩"作为状态指标，必须按对手质量分级：

| 主场对手等级 | 输球的信号意义 |
|------------|--------------|
| 联赛前 3 强队（榜首集团） | **输给强队 ≠ 主场能力崩溃**（不可作为支持下盘的信号） |
| 同级别中游队 | 输给同级别对手 = 主场能力真实低迷 |
| 联赛后 4 弱队 | 输给弱队 = 主场严重崩盘（强信号） |

## 权重调整

| 主场近期输球对手构成 | "主场崩溃"权重 | 说明 |
|------------------|-------------|------|
| 主要输给前 3 强队（≥ 60%） | **15%** | 输强队是正常结果 |
| 主要输给同级别队（≥ 60%） | **30%** | 真实主场能力低迷 |
| 主要输给弱队（≥ 40%） | **40%** | 主场严重崩盘 |

## Why

输给联赛前 3 强队（榜首集团）是**实力差距的正常体现**，不代表主场能力崩溃。这些输球不应被用来支持"主场不行"的判断，尤其当本场对手是中下游球队时。

相反，如果主场输给同级别或更弱的对手，才说明主场能力真实低迷。

## How to apply

1. 统计主队近 3~5 场主场输球记录
2. 查询每场输球对手的联赛排名
3. 按对手质量分级：前 3 强队 / 同级别队 / 弱队
4. 计算各级别占比
5. 若输给前 3 强队占比 ≥ 60%，"主场崩溃"权重下调至 15%
6. 配合 [[feedback_h2h_home_dominance_same_direction]] 使用，H2H 主场优势权重上调至 35%

## 案例

Chelsea 2-1 Tottenham Hotspur（2026-05-20 英超）：Chelsea 主场近 3 场全败（0-3 诺丁汉、0-1 曼联、0-3 曼城），其中 0-3 曼城（第 2）、0-1 曼联（第 3）、0-3 诺丁汉（当时状态火热，近 5 场 4 胜），**输给强队占主导**，误将"主场崩溃"权重设为 40%，实际应降至 15%。
