## 一、欧亚同步升盘的优先级判定

### 教训来源
Chelsea 1-0 Leeds United（2026-04-26 足总杯半决赛）

### 错误判断
将升盘（平半→半球）判定为"诱上"，最终推荐下盘（利兹），实际上盘（切尔西）1-0赢球赢盘。

### 核心规则

**规则 1：欧亚同步升盘 = 真实定价，非诱盘**

当以下条件同时满足时，升盘必须视为庄家真实看好上盘：
- 欧赔主胜方向持续下降（如 2.12 → 2.05）
- 亚盘同步升盘（如平半 → 半球）
- 两个方向一致且在早中期完成（非临场突变）

此时不可套用军规 #4（开大+不变盘+诱上）或 #5（升盘诱上），因为升盘本身是定价修正。

**规则 2：升盘前凯利极低信号不可被升盘后数据覆盖**

- 若升盘前主凯利持续 < 0.97（尤其 < 0.90），说明庄家初盘阶段就真实看好主队
- 升盘后凯利回升至 1.0+ 是盘口加深后的数学结果，不代表庄家态度反转
- 正确做法：以升盘前的凯利趋势为"底色"，升盘后的凯利变化仅作微调参考

**规则 3：反常识升盘 = 最强信号**

当基本面明显不利（如连败、状态崩溃）但庄家仍然升盘时：
- 这是庄家掌握超越公开信息的判断（阵容、战术、杯赛动力等）
- 越是与基本面矛盾的盘口变动，越可能包含内幕信息
- 此时基本面权重必须大幅下调，盘口信号权重上调

## 二、即时节点凯利异动的正确解读

### 错误模式
即时节点（封盘前最后一次调赔）出现主胜凯利跳升 + 客胜凯利骤降 → 误判为"客胜信号增强"。

### 正确解读

即时节点的凯利突变需区分两种情况：

| 情况 | 特征 | 正确解读 |
|------|------|----------|
| 趋势延续型 | 凯利变化方向与前 3~4 个节点一致，幅度递增 | 真实信号，可信 |
| 突变反转型 | 凯利方向与前序节点相反，或幅度异常（单节点变化 > 前序总变化） | 庄家封盘前诱导操作，**反向解读** |

本案即时节点：
- 临场①→②→③：主凯利 1.015 → 1.025 → 1.027（缓慢上升，幅度 +0.012）
- 临场③→即时：主凯利 1.027 → 1.053（单节点 +0.026，超过前序总变化）
- 同时平局赔率跳升 0.34 — 属于**突变反转型**，应反向解读为诱下操作

### 判定规则

> 即时节点单项凯利变化幅度 > 前序 2~3 个节点累计变化时，该异动为庄家封盘诱导，应反向解读或忽略。

## 三、基本面与盘口信号的权重分配

### 错误模式
联赛状态崩溃（5连败）→ 先入为主认为"球队不行" → 将升盘解读为诱盘 → 忽视盘口传递的真实信号。

### 正确权重规则

| 场景 | 基本面权重 | 盘口信号权重 | 说明 |
|------|-----------|-------------|------|
| 基本面与盘口方向一致 | 40% | 60% | 正常分配 |
| 基本面与盘口方向矛盾（盘口加深） | 20% | 80% | 庄家可能有非公开信息 |
| 杯赛 + 基本面矛盾 | 15% | 85% | 杯赛状态独立于联赛 |
| 基本面碾压 + 盘口浅开 | 30% | 70% | 参考"碾压级基本面"规则 |

### 杯赛特殊性

- 杯赛（尤其淘汰赛/半决赛）的球队状态与联赛可以完全脱节
- 联赛连败不等于杯赛连败，两条线的动力、轮换、战术完全不同
- 主场杯赛半决赛/决赛存在"杯赛DNA"加成，历史交锋权重应上调

## 四、"阻上"形态的严格判定标准

### 错误模式
上盘水位 2.08（半球盘）→ 判定为"高水阻上" → 推演上盘难出。

### 正确标准

阻上型盘路的核心特征是**单方向持续上升**（参考 reference_dynamic_analysis.md 1.2 节）。

仅凭单一水位值不能判定阻上，必须满足：
1. 水位从某个起点**持续单向攀升**，升幅 > 10 个点（如 0.92 → 1.05）
2. 中间不能有明显的升降反复震荡
3. 升盘后水位重置不算"攀升"（升盘本身会重置水位基准）

本案实际情况：
- 升盘后主水 2.07 → 2.02 → 2.06 → 2.08（先降后升，震荡幅度仅 6 个点）
- 这是**正常波动**，不构成阻上形态
- 半球盘 2.08 属于中高水，非极端高水（极端高水应 > 2.15）

### 升盘后水位的正确解读

升盘后水位需要重新建立基准：
- 升盘后上盘高水 + 下盘低水 + 不再变盘 → 可能是军规 #1（阻上诱下，上盘胜）
- 升盘后上盘水位持续走高（2.05 → 2.10 → 2.15+）→ 才是真正的给水型/阻上存疑
- 升盘后水位小幅震荡（±3~5 个点）→ 正常调整，不构成形态判断依据

## 五、连续降盘的正确解读与军规匹配

### 教训来源
浙江 2-1 深圳新鹏城（2026-05-02 中超）

### 错误判断
初盘开大（-0.75 vs 理论 -0.50）+ 上盘持续受热（主凯利 < 0.90）+ 临场连续降盘（-0.75→-0.50→-0.25），误套军规 #4（诱上→下盘胜或平），推荐下盘（深圳不败）。实际浙江 2:1 主胜赢盘。

### 核心规则

**规则 1：降盘 = 降低上盘赢盘门槛 = 对上盘有利**

降盘的本质是减少上盘需要覆盖的让球深度，使上盘更容易赢盘。不可将降盘误读为"庄家不看好上盘"。如果庄家真的不看好上盘，应该升盘（加深让球）或维持不变。

| 操作 | 含义 | 对上盘影响 |
|------|------|-----------|
| 降盘（如 -0.75→-0.50→-0.25）| 降低让球深度 | **有利**（更容易赢盘）|
| 升盘（如 -0.50→-0.75→-1.0）| 加深让球深度 | **不利**（更难赢盘）|
| 不变盘 | 维持现有让球 | 需结合水位判断 |

**规则 2：初盘开大 + 连续降盘 → 军规 #3（转场景三），不是军规 #4**

| 场景 | 正确匹配 | 错误匹配 |
|------|---------|---------|
| 初盘开大 + 不变盘 + 上盘大热 + 降水至低水 | 军规 #4（诱上→下盘胜）| — |
| 初盘开大 + 早中期/临场降盘至合适 | **军规 #3（修正→转场景三）**| ~~军规 #4~~ |
| 转场景三后 + 临场再降盘 | **军规 #10（减小上盘赔付→上盘赢球赢盘）**| — |

关键区分：军规 #4 的前提是"不变盘"，一旦发生降盘，应走军规 #3 路径。连续降盘更是强烈的定价修正信号，说明庄家认为初盘高估了上盘让球深度。

**规则 3：欧赔主胜上升 + 亚盘降盘 = 同步定价修正（"上盘小胜"信号）**

当欧赔主胜赔率上升（如 1.74→2.26）与亚盘降盘（如 -0.75→-0.25）同步发生时：
- 这不是"主胜不被看好"，而是"主胜赢球但净胜球少"
- 欧赔放开主胜赔付 + 亚盘降低让球门槛 = 庄家认为上盘小胜（1 球）
- 与"欧亚同步升盘 = 真实定价"（第一章）互为镜像：**欧亚同步降盘 = 真实定价修正**

**规则 4：Pinnacle 全面偏移时凯利间距 < 0.05 必须严格忽略**

本案 Pinnacle 欧赔三项凯利全程 > 1.0（全面偏移庄家），即时节点间距仅 0.023（主胜 1.050 vs 平局 1.027），远小于 0.05 阈值。分析时违反 reference_kelly_index.md 2.1b 节规则，仍将 Pinnacle 凯利差异作为主要依据，导致错误强化"平局信号"。

## 六、近况崩溃期的主场反弹规律

### 教训来源
同上（浙江 2-1 深圳新鹏城）

### 错误判断
浙江近 4 场 0 进球（状态崩溃）→ 过度降低主胜置信度。深圳客场同样糟糕，误判为两队都差→平局概率最高。

### 核心规则

**规则 1：连败队主场 vs 客场低迷队 = 主场反弹概率高**

| 条件 | 判断 |
|------|------|
| 主队连续 3 场+ 不胜 + 客队客场连续 2 场+ 不胜 | **主场方反弹概率 > 客场方** |
| 主队连续 3 场+ 不胜 + 客队客场状态正常 | 需综合判断 |
| 两队近况均崩溃 | 主场方反弹概率更高（主场球迷+排名压力+训练环境优势） |

低迷队主场比赛承受更大反弹压力（主场球迷+保级/排名压力），而客场低迷队缺乏反弹动力和环境。

**规则 2：历史交锋在低迷期权重应上调而非下调**

| 条件 | 权重调整 |
|------|---------|
| 主队近况好 + 历史交锋占优 | 正常权重 |
| 主队近况差 + 历史交锋占优 | **权重上调**（低迷期需要"有把握的对手"止颓）|
| 主队近况差 + 历史交锋不占优 | 权重正常或下调 |

**规则 3：近况对手质量必须量化**

不可简单统计"近 N 场胜率"作为状态指标，必须考虑对手实力：
- 输给榜首/前 3 的比赛不等于输给同级别对手
- 连续输给强队后遇到弱队/中游队 → 反弹概率上升
- 应区分"实力输球"（对手太强）和"状态输球"（对同级别对手也输）

## 七、临场末端急剧降盘 + 主水反升 ≠ 军规 #10

### 教训来源
霍芬海姆 3-3 斯图加特（2026-05-02 德甲）

### 错误判断
初盘合适（平手盘符合欧赔 2.37 理论），临场④（开球前 30 分钟）盘口从平手突降至 -0.25，欧赔主胜同步下降 2.48→2.28，判定为军规 #10（临场降盘 = 真实看好上盘），推荐主队上盘。实际 3:3 平局，主队让 -0.25 输盘。

### 核心规则

**规则 1：军规 #10 的水位方向验证（必检项）**

军规 #10（初盘合适 + 临场降盘 = 减小上盘赔付，真实看好上盘赢球赢盘）有一个隐含前提：降盘后主队水位应**下降或维持**（庄家在收紧上盘赔付）。

| 降盘后主水变化 | 含义 | 是否符合 #10 |
|--------------|------|-------------|
| 主水下降或维持（如 1.88→1.80） | 庄家降低上盘赔付门槛，真实收紧 | **符合 #10，上盘看好** |
| 主水上升（如 1.88→2.01） | 庄家加大上盘赔付敞口，赔付压力增大 | **不符合 #10，需重新评估** |

本案：临场③主水 1.88 → 临场④降盘至 -0.25 后主水跳升至 2.01（升 13 个点）。降盘的同时主水反升 = 庄家并未真正收紧上盘赔付，而是在降盘的表象下扩大了上盘赔付敞口。

**规则 2：临场末端急剧变盘（开球前 ≤ 30 分钟）的特殊处理**

| 变盘时机 | 变盘幅度 | 正确处理 |
|---------|---------|---------|
| 中盘阶段降盘（-24h 以上） | 任意 | 按标准军规匹配 |
| 临场①②降盘（-6h ~ -3h） | ≤ 0.25 球 | 按标准军规匹配 |
| 临场③④降盘（-1.5h 以内） | **≥ 0.25 球** | **高风险异动**，不可直接套用 #10，须验证水位方向 |
| 临场④降盘（-0.5h 以内） | **≥ 0.25 球** | **知情资金窗口异动**，置信度额外扣减 10 分 |

临场末端的急剧变盘（尤其 ≥ 0.25 球）可能是知情资金窗口的诱导操作，不可等同于早中期的定价修正。

**规则 3：欧赔下降 + 亚盘降盘 + 主水反升 = "制造主队被看好假象"**

当三个信号同时出现：
- 欧赔主胜赔率下降（表面看好主队）
- 亚盘降盘（表面增加主队让球深度）
- 主队水位反升（实际增大主队赔付敞口）

此组合是一种隐蔽的诱上手法：前两个信号吸引资金涌入主队，第三个信号（主水反升）暴露了庄家真实意图——庄家在主队方向的赔付敞口增大，说明庄家不惧怕主队打出。

> 对比真实 #10 信号：欧赔主胜下降 + 亚盘降盘 + 主水**下降或维持** = 庄家在三个维度同步收紧主队方向赔付，才是真实看好主队。

## 八、历史交锋高平局率对结论的强制修正

### 教训来源
同上（霍芬海姆 3-3 斯图加特）

### 错误判断
近 4 个赛季 5 次交锋 4 次平局（80% 平局率），但最终结论仍以主胜为首选，将平局高概率仅列为"风险提示"而非"核心依据"。

### 核心规则

**规则 1：H2H 平局率 ≥ 60% 时平局必须进入核心依据**

| H2H 近 5~10 场平局率 | 处理方式 |
|---------------------|---------|
| < 40% | 正常参考权重 |
| 40%~59% | 平局作为次选列入结论 |
| **≥ 60%**（如 5 场中 ≥ 3 场平局） | **平局必须作为首选或与首选并列**，不可仅列为"风险提示" |
| **≥ 80%** | 平局作为首选，其他结果降为次选 |

**规则 2：盘口信号与 H2H 高平局率的权重平衡**

当盘口信号指向某方胜出，但 H2H 平局率 ≥ 60% 时：
- 盘口信号权重：50%
- H2H 平局率权重：50%
- 置信度上限：70（不可给出 > 70 的置信度）

本案：军规 #10 指向主胜，但 H2H 80% 平局率构成强对冲。正确结论应为平局首选 / 主胜次选，置信度不超过 70。

**规则 3：同积分 / 同排名对手之间平局概率天然偏高**

当两队满足以下条件时，平局基础概率上调：
- 积分差 ≤ 3 分
- 排名相邻（差 ≤ 2 位）
- 均有明确赛季目标（如同争欧战 / 同保级）

此时即使盘口信号有方向性，平局也应作为强备选。

**规则 4：B/C 冲突中平局凯利报警的优先级**

当算法 B 推演为某方胜出，但算法 C 中有公司平局凯利 < 0.96 报警：
- 这是 B/C 冲突，**不可定性为"基本一致"**
- 必须降低置信度 10~15 分
- 平局必须至少作为次选进入最终结论

本案：Tipico 平局凯利 0.935（< 0.96），构成对军规 #10（主胜）的明确冲突，却被定性为"B/C 基本一致"，导致平局权重被严重低估。

## 九、水位升降方向与热度判断

### 教训来源
Newcastle United 3-1 Brighton（2026-05-02 英超）

### 错误判断
将主队亚盘水位 1.90→2.05（上升）误判为"主队大热"，实际主队是冷门。基于错误的热度方向匹配军规 #5（诱上→下盘胜），实际应匹配军规 #1（初盘开大+不变盘+下盘大热→诱下→上盘胜）。预测平局/客胜（置信度88），实际主队3-1大胜。

### 核心规则

**规则 1：水位变化方向与热度的正确对应**

| 水位变化 | 含义 | 热度 |
|---------|------|------|
| 某方水位**下降**（如 2.01→1.88） | 资金涌入该方，庄家降低赔付 | **该方大热** |
| 某方水位**上升**（如 1.90→2.05） | 该方投注不足，庄家提高赔付吸引 | **该方冷门** |

亚盘与欧赔必须交叉验证：
- 亚盘水位下降方 = 热门方
- 欧赔赔率下降方 = 热门方（如客胜赔率 2.62→2.55 = 客队热门）
- 两者方向必须一致

本案：亚盘布莱顿水位 2.01→1.88（下降=热门），欧赔客胜 2.62→2.55（下降=热门），**布莱顿（下盘）才是大热方**。

**规则 2：热度判断必须在军规匹配之前完成并单独验证**

| 步骤 | 必须动作 |
|------|---------|
| 判断热度前 | 列出亚盘双方水位变化方向 + 欧赔三项赔率变化方向 |
| 得出热度结论后 | 用一句话明确写出"热门方=X，证据：水位从Y降至Z" |
| 匹配军规前 | 再次核对热度方向是否与选定的军规前提一致 |

**规则 3：Betfair 交易所赔率变化 = 市场热度指标（非庄家意图）**

Betfair 某项赔率下降 = 散户资金涌入 = 该方向大热。不可按传统庄家逻辑（"赔率下降=庄家看好"）解读。

本案：Betfair 客胜从 2.74 降至 2.62（散户押注布莱顿）= 布莱顿大热信号，进一步印证布莱顿（下盘）是热门方。

**规则 4：Pinnacle 全面偏移时凯利间距 < 0.05 必须忽略**

（同第五章规则 4。本案 Pinnacle 欧赔三项凯利 1.036/1.009/1.024，全 > 1.0，间距 0.027 < 0.05，应忽略但被当作"平局易打出"的依据。）

**规则 5：保级队主场动力权重上调**

当主队处于保级区（排名 15-18）且主场作战时：
- 近况（连败/进攻低迷等）权重下调至 15%
- 保级战意 + 主场优势权重上调至 25%
- 盘口信号权重维持 60%

近况对手质量必须量化：输给前3强队 ≠ 状态崩溃。

## 十、欧亚同步升盘规则的边界条件（防止教条化）

### 教训来源
云南玉昆 1-2 浙江队（2026-05-06 中超）

### 错误判断
临场④升盘 -0.25→-0.5 + 欧赔主胜同步降 2.08→1.98 + 客胜抬升 2.93→3.21，机械套用"欧亚同步升盘=真实定价"规则推荐上盘（云南玉昆）。实际云南 1-2 输球，上盘输盘。

### 核心规则

**规则 1：欧亚同步升盘的"真实定价"判定必须含即时节点验证**

"欧亚同步升盘=真实定价"是强信号，但有边界条件。判定真实定价必须同时满足：

| 验证项 | 真实定价 | 诱导操作 |
|--------|---------|---------|
| 升盘时机 | 早中期完成（中盘②或更早） | **临场③④（≤1.5h 内）才升盘** |
| 升盘后欧赔 | 继续同方向或稳定 | **即时节点出现欧赔回调**（主胜回升/客胜回调）|
| 升盘后主水 | 适度调整（±5 点内）或下降 | **主水大幅反升 ≥ 10 点** |
| 升盘幅度 | ≤ 0.25 球 | ≥ 0.25 球（在临场④） |

本案违规点：
- 临场④（开球前 30 分钟）才升盘 → 不属于"早中期完成"
- 即时节点 Pinnacle 主胜 1.98→**2.01**（回升），客胜 3.21→**3.20**（回调）→ 信号方向反转
- 临场③主水 1.87 → 临场④主水 **1.99**（+12 点）→ 即时再升至 **2.02**（再 +3 点）→ 主水持续反升

**规则 2：升盘 + 主水持续上升 = 升盘版"诱上"（与案例 7 镜像）**

参考第七章（霍芬海姆 3-3 斯图加特）"降盘 + 主水反升 ≠ 军规 #10"的镜像规则：

| 操作组合 | 表面信号 | 实质 |
|---------|---------|------|
| 降盘 + 主水反升 | 看似看好上盘 | 庄家扩大上盘赔付敞口（案例 7） |
| **升盘 + 主水反升** | 看似真实定价修正 | **庄家在升盘时扩大上盘赔付敞口（本案）** |

判定规则：
- 升盘后主水变化 ≤ 5 点 = 正常修正
- 升盘后主水上升 6~10 点 = 中性观察
- **升盘后主水持续上升 ≥ 10 点（含即时延续上升）= 诱上信号**

**规则 3：临场④升盘必须独立验证（不可直接套用早中期规则）**

临场④（开球前 ≤ 30 分钟）属于知情资金窗口，此时的升盘行为与早中期升盘性质不同：

| 升盘时机 | 性质 | 处理规则 |
|---------|------|---------|
| 初盘②~中盘② | 定价基准修正 | 直接套用"欧亚同步升盘=真实定价" |
| 中盘③~临场① | 消息消化修正 | 套用"欧亚同步升盘=真实定价"，置信度扣减 5 |
| 临场②~③ | 资金调整 | 必须验证主水方向 + 即时节点延续性 |
| **临场④（≤30min）** | **知情资金窗口** | **不可直接套用真实定价规则；必须三重验证（欧赔延续/主水方向/客胜凯利）** |

**规则 4：客胜 Pinnacle 凯利 < 0.96 是被忽视的反向信号**

主胜方向被多家公司轻度压制（凯利 0.99~1.0）容易吸引注意，但客胜方向单家公司凯利接近报警阈值往往是更隐蔽的真实信号：

| 信号组合 | 解读 |
|---------|------|
| 主胜凯利 0.99~1.0（多公司） + 客胜凯利 > 1.0（多公司） | 主胜被压制，可信 |
| **主胜凯利 0.99~1.0（多公司） + Pinnacle 客胜凯利 < 0.96** | **客胜真实信号被掩盖；主胜压制可能是吸引上盘资金的诱导** |
| 主胜凯利 < 0.96（多公司） + 客胜凯利 > 1.0（多公司） | 主胜真实看好 |

判定规则：
- Pinnacle 任一方向凯利 < 0.96 必须列入核心信号清单（不可仅列为辅助）
- 当 Pinnacle 客胜凯利 < 0.96 与"主胜被多家轻度压制"并存时，**优先级反转**——主胜压制视为诱导，客胜低凯利视为真实

**规则 5：中超球队主场优势权重应下调**

中超主场优势的不确定性显著高于欧洲五大联赛：

| 联赛 | 主场胜率基准 | 排名差权重 |
|------|------------|-----------|
| 五大联赛 | 45%~50% | 高（排名差 ≥ 5 名 = 强信号）|
| **中超** | **38%~42%** | **中（排名差 ≥ 5 名仅作为参考信号）**|

原因：
- 中超球队赛季密集（杯赛+联赛+亚冠）+ 体能管理差 → 状态稳定性低
- 球员阵容深度有限 → 单场伤停影响放大
- 主场草皮/天气/球迷优势在中超被冲淡

应用：
- 中超比赛"主场+排名差"组合置信度上限 75（非 80+）
- 主场近期防守崩盘前科（如 0-3、0-4 大败）必须显著扣减置信度 10 分
- 客场连败队的"反弹潜力"权重应纳入考虑（不可只看绝对客场战绩）

### 与既有规则的关系

| 规则 | 关系 |
|------|------|
| 第一章（欧亚同步升盘=真实定价） | **本章为该规则的边界条件补充，非否定** |
| 第七章（降盘+主水反升 ≠ 军规 #10） | **本章为其镜像（升盘版）** |
| 第五章规则 4（Pinnacle 凯利间距 < 0.05 忽略） | 本章规则 4 为补充：单方向 < 0.96 应作为核心信号 |

## 十一、全程不变盘 + 长期给水 + 临场④协同压主胜 = 诱上型变体

### 教训来源
Shanghai SIPG FC 2-2 Zhejiang（2026-05-15 中超）

### 错误判断
全程 10 节点盘口固定 -0.5 不变，临场①~③ 主水从 1.93 抬升至 2.02（给水），临场④四公司协同压主胜（Pinnacle 2.01→1.84、Marathon 1.93→1.79、William 1.95→1.83、Betfair 2.02→1.89）+ Pinnacle 主胜凯利从 1.046 突降至 1.015。机械套用"临场④多公司协同压主胜=真实资金信号"逻辑，推荐上盘+主胜，置信度 65。实际 2-2 平局，半球盘海港让 -0.5 输盘。

### 核心规则

**规则 1：全程不变盘 + 长期给水 + 临场④突压 = 诱上型完整路径**

诱上型经典识别在 `reference_dynamic_analysis.md` 1.3 节描述为"长期低水 + 临场超低水"，本案揭示的变体路径为：

| 阶段 | 操作 | 散户心理 |
|------|------|----------|
| 初盘~中盘 | **盘口锁定不变**（如全程 -0.5） | "庄家很有把握 → 上盘稳" |
| 临场①~③ | **持续给水抬高主水**（如 1.93→2.02，给水 9 点） | "赔付变高 → 押主队更划算" |
| 临场④ | **多公司协同突压主胜赔率/水位** | "知情资金涌入 → 主胜要出了" |
| 即时 | **小幅回调暴露诱导意图** | （散户已上钩） |

判定要件（全部满足即视为诱上型变体）：
- 全程 ≥ 8 节点盘口不变盘
- 临场①~③ 给水累计 ≥ 7 点
- 临场④ 多公司（≥ 3 家）协同压主胜，幅度 ≥ 0.10
- **即时节点出现反向小回调**（哪怕仅 3 点）

**规则 2：即时节点小幅回调是诱上的核心识别信号（防止"真实资金"误判）**

临场④若为真实知情资金涌入，即时节点应满足：
- 主水继续下降或维持（不可上升）
- 欧赔主胜继续下降或维持（不可回升）
- 凯利信号继续延续方向

若即时节点出现以下任一信号 = 诱上嫌疑：

| 即时节点信号 | 真实定价 | 诱上嫌疑 |
|-------------|---------|---------|
| 主水变化 | 继续下降 ≥ 0 | **回升 ≥ 3 点** |
| 欧赔主胜 | 继续下降 ≥ 0 | **回升 ≥ 0.02** |
| Pinnacle 主胜凯利 | 维持或继续下降 | **回升** |

本案违规点：即时主水 1.85→1.88（+3 点），欧主胜 1.84→1.87（+0.03），主胜凯利 1.015→1.021（回升）→ 三项同时回调 = 诱上确认。

**规则 3：Pinnacle 客胜凯利 > 1.05 在协同压主胜场景下作为反向核心信号（镜像规则）**

与 [[feedback_kelly_priority_inversion]]（客胜 < 0.96 + 主胜多家压制 → 主胜压制视为诱导）互为镜像：

| 信号组合 | 解读 |
|---------|------|
| Pinnacle 客胜凯利 < 0.96 + 多家压制主胜 | 主胜压制视为诱导，客胜真实看好 |
| **Pinnacle 客胜凯利 > 1.05 + 多家协同压主胜** | **主胜压制视为诱导，客胜放水视为庄家不惧客胜打出** |

凯利 > 1.05 的语义解读：
- 字面：Pinnacle 客胜赔率比市场平均高 5%+
- 实质：庄家在客胜方向开放高赔付敞口
- 当与"多家协同压主胜"并存时，**不可解读为"客胜被抛售"**，而是庄家**敢于让客胜资金进入**——这是反向支持客胜的信号

判定规则：
- Pinnacle 任一方向凯利 > 1.05 时，必须列入核心信号清单
- 与"协同压主胜"并存时，触发优先级反转：协同压主胜视为诱导，客胜放水视为庄家不惧客胜

**规则 4：本赛季排名+积分反转 H2H 优势时，当下指标权重高于历史**

| 条件 | 权重 |
|------|------|
| 主场 H2H 占优 + 本赛季排名同向占优 | H2H 权重正常（核心依据）|
| **主场 H2H 占优 + 本赛季排名反向（H2H 占优方排名更低/积分更少）** | **本赛季排名权重 > H2H 权重；置信度至少扣减 10 分；平局须列为强备选** |
| H2H 不占优 + 本赛季排名占优 | 当下排名为核心依据 |

本案：海港主场对浙江 H2H 3胜1平1负（占优），但海港本赛季排名第 12（7 分）低于浙江第 8（10 分）→ 本赛季排名反向，H2H 不可作为核心依据。

**规则 5：中超主队近期主场负于强队 → 让球能力存疑（半球+ 盘口扣减置信度）**

中超主队近期主场记录中存在以下情况之一，半球及以上让球盘必须额外扣减置信度 5 分：
- 1-2 / 0-1 / 0-2 等小负于联赛前 3 强队
- 主场让球盘连续 2 场+ 未兑现（输盘或走盘）

原因：中超主队让球能力波动远大于欧洲五大联赛，近期主场负盘记录暗示本场让球能力存疑。

**规则 6：客队"绝对客场战绩"必须按对手质量分级**

不可简单统计"客场 N 战 X 胜"作为客队状态指标，必须按对手分级：

| 客场对手等级 | 输球的信号意义 |
|------------|--------------|
| 联赛前 4 强队（榜首集团）| **输给强队 ≠ 状态崩溃**（不可作为支持本场上盘的信号） |
| 同级别中游队 | 输给同级别对手 = 状态真实低迷 |
| 联赛后 4 弱队 | 输给弱队 = 客场严重崩盘（强信号）|

本案：浙江客场 5 战 1 胜 1 平 3 负，但 3 负中有 0-4 成都（第 1）、0-3 大连（第 3）、0-2 武汉（弱队但客场）→ 输给榜首集团占主导，"客场极弱"判断不成立；对阵海港（排名 12）属于"同级别或弱于自身"，反弹潜力被严重低估。

## 十二、强基本面碾压场景下"诱上变体"信号的过滤

### 教训来源
AS Roma 2-0 Lazio（2026-05-17 意甲第37轮，罗马德比）

### 错误判断
临场④ Pinnacle 主水从 1.89 反升至 1.96（+7 点）+ 欧赔主胜从 1.52 升至 1.55 + 即时节点主水回落至 1.93 + 欧赔即时回调 1.55→1.53，机械套用 [[feedback_lure_up_variant]]（诱上型变体）+ 第十一章规则 1~2，判定为"诱上信号"推荐下盘（拉齐奥 +1）。实际罗马 2-0 主场胜，**上盘赢球赢盘**，亚盘方向完全错误。

### 核心规则

**规则 1：碾压级基本面下，临场④主水反升 = 风控收紧（非诱上）**

当满足以下基本面条件时，临场④的主水反升不可视为诱上信号，而应解读为庄家在散户大量押注上盘时的**风控收紧**（提高主水以平衡赔付敞口）：

| 基本面要素 | 阈值 |
|-----------|------|
| 联赛积分差 | **≥ 15 分** |
| 客队近 3 场战绩 | **2 场+ 大败（净负 ≥ 2 球）** |
| 客队赛程压力 | **3~7 天内有 2 场+ 重要比赛（如杯赛+联赛连战）** |
| 主队主场近况 | **近 3 个主场全胜 + 进 5 球+** |
| 历史交锋 | **近 5 场主队胜率 ≥ 60%（不含走盘）** |

满足上述 ≥ 3 项 = **碾压级基本面**，此时临场④主水反升 ≤ 10 点应视为风控收紧，不可触发诱上识别。

本案匹配项：
- 积分差 16 分（罗马 67 vs 拉齐奥 51）✓
- 拉齐奥 3 天前杯决 0-2 国米 + 4 天前联赛 0-3 国米（连续大败）✓
- 拉齐奥赛程压力极大（杯决+联赛+德比连战）✓
- 罗马近 4 个主场进 8 球（4-0 佛罗伦萨、3-0 比萨、1-0 莱切）✓
- H2H 近 5 场罗马 3 胜 1 平 1 负 ✓
- **5 项全部满足** → 碾压级基本面成立 → 临场④主水反升 7 点应视为风控收紧

**规则 2：诱上型变体识别必须排除"碾压级基本面"先决条件**

[[feedback_lure_up_variant]] 与第十一章规则 1（全程不变盘+给水+临场④协同压主胜=诱上变体）的应用必须满足前置条件：

| 前置条件 | 满足时 | 不满足时 |
|---------|-------|---------|
| 基本面均衡或矛盾（积分差 < 10 分 / 客队状态正常 / H2H 接近） | 触发诱上变体识别 | — |
| **基本面碾压（满足规则 1 ≥ 3 项）** | — | **优先视为风控收紧，不触发诱上识别** |

判定优先级（自上而下）：
1. **基本面碾压** → 反弹概率 / 风控收紧 → 上盘真实看好
2. 基本面均衡 + 全程不变盘 + 给水 + 临场④协同压主胜 → 诱上变体
3. 基本面均衡 + 临场④升盘 + 主水反升 → 升盘版诱上（第十章）

**规则 3：临场④主水反升幅度的分级判定**

主水反升的诱上嫌疑必须按幅度分级，不可"任何反升均视为诱上"：

| 反升幅度 | 性质 | 处理 |
|---------|------|------|
| ≤ 5 点 | 正常市场波动 | 忽略 |
| **6~10 点** | **风控调整或诱上**（需结合基本面判定）| **碾压级基本面 → 风控；均衡基本面 → 诱上嫌疑** |
| 11~15 点 | 诱上嫌疑较高 | 即使有强基本面，置信度扣减 5 分 |
| ≥ 16 点 | 诱上确认 | 不论基本面，警惕诱上 |

本案临场④主水反升 7 点（1.89→1.96），即时回落 3 点至 1.93 → 净反升 4 点（小幅波动），**远未达到诱上确认阈值**。

**规则 4：欧赔同步小幅回升 ≤ 0.03 在碾压级基本面下不构成诱上确认**

[[feedback_lure_up_variant]] 中"欧赔即时回调"作为诱上确认信号有边界：

| 欧赔即时回调幅度 | 碾压级基本面 | 均衡基本面 |
|---------------|-----------|-----------|
| 0.01~0.03 | **正常市场调整，忽略** | 诱上嫌疑信号 |
| 0.04~0.07 | 中性观察，结合主水方向 | 诱上确认信号 |
| ≥ 0.08 | 即使有强基本面也警惕 | 诱上强确认 |

本案 Pinnacle 主胜临场④→即时回调 0.02（1.55→1.53），在碾压级基本面下应视为正常调整。

**规则 5：客队连续大败 + 短期内重要比赛后的"心理疲态"权重**

当客队满足以下条件时，"疲态系数"必须显著上调（作为支持主场上盘的核心权重）：

| 疲态信号 | 权重影响 |
|---------|---------|
| 距上场重要比赛 ≤ 3 天 | 体能扣减客队战力 15% |
| 上场重要比赛失利（如杯决败北） | 心理扣减客队战力 10% |
| 近 2 场失球 ≥ 5 个（防线崩塌） | 防守效率扣减 20% |
| 客场连战 | 长途+客场环境扣减 10% |

本案拉齐奥触发前 3 项：
- 距杯决 0-2 国米仅 3 天 → 体能 -15%
- 杯决决赛失利 + 联赛 0-3 → 心理 -10%
- 近 2 场失 5 球 → 防守 -20%
- **综合战力扣减 ≈ 35%**，本案应视为客队"残血出战"

此时即使盘口出现轻微诱上信号，也应优先采信基本面的"上盘碾压"判断。

**规则 6：罗马德比/同城德比"小球+1-0"惯性 ≠ 走盘风险**

德比战常见 1-0 / 0-1 / 1-1 小比分，但分析时不可机械套用"走盘风险"扣减上盘置信度：

| 场景 | 处理 |
|------|------|
| 同城德比 + 双方实力接近 + 同积分区 | 走盘风险确实较高，扣减上盘置信度 5~10 分 |
| **同城德比 + 实力碾压（积分差 ≥ 15 分）+ 客队疲态** | **走盘惯性被打破，按碾压逻辑预判 2 球+ 比分** |

本案罗马 vs 拉齐奥积分差 16 分 + 客队 35% 战力扣减 → 不属于"实力接近的德比"，走盘风险显著低于普通德比，最终 2-0 比分印证主胜碾压逻辑。

### 与既有规则的关系

|------|------|
| 第十一章规则 1（全程不变盘+给水+临场④协同压主胜=诱上变体） | **本章为其前置条件补充**：必须先排除碾压级基本面 |
| [[feedback_lure_up_variant]]（诱上型变体记忆） | **本章为该记忆的边界条件** |

## 十三、浅盘升盘诱上组合识别（初盘开小 + 临场升盘 + 主水大幅反升）

### 教训来源
浙江 4-1 山东泰山（2026-05-20 中超）

### 错误判断
初盘开小（0.25 < 理论 0.5）+ 临场②升盘至 0.0 + 欧赔主胜同步下降 2.92→2.65，机械套用"欧亚同步升盘=真实定价"规则推荐客队（山东泰山）。实际浙江 4:1 主场大胜，上盘赢球赢盘，预测方向完全错误。

### 核心规则

**规则 1：浅盘升盘诱上组合的五项识别要件**

当以下条件**全部满足**时，必须识别为"浅诱 + 升盘版诱上"双重组合，真实看好下盘（主队）：

| 要件 | 阈值 | 本案数据 |
|------|------|---------|
| ① 初盘开小 | 实际盘口 < 理论盘口 0.25 球+ | 0.25 < 0.5（开小 0.25） |
| ② 临场②~③升盘 | 升幅 ≥ 0.25 球 | 0.25→0.0（升 0.25） |
| ③ 升盘后主水持续反升 | ≥ 15 点（含临场③延续） | 1.83→2.01→2.08（+25 点） |
| ④ 即时节点欧赔回调 | ≥ 0.05 | 2.65→2.75（+0.10） |
| ⑤ 升盘后亚盘主凯利虚高 | 持续 > 1.08 | 1.083→1.127→1.085→1.118 |

**判定逻辑**：
- 初盘浅开（要件①）= 浅诱（诱导上盘资金）
- 临场②升盘（要件②）= 制造"庄家看好上盘"假象
- 主水大幅反升（要件③）= 庄家在升盘时扩大上盘赔付敞口（与"真实定价"矛盾）
- 即时欧赔回调（要件④）= 封盘前暴露真实意图（不看好上盘）
- 主凯利虚高（要件⑤）= 庄家不惧上盘打出

**规则 2：与"欧亚同步升盘=真实定价"的边界区分**

| 场景 | 升盘后主水变化 | 即时欧赔 | 主凯利 | 判定 |
|------|--------------|---------|--------|------|
| 真实定价（第一章） | ≤ 10 点或下降 | 延续下降或稳定 | < 1.05 | 真实看好上盘 |
| **浅盘升盘诱上（本章）** | **≥ 15 点持续反升** | **回调 ≥ 0.05** | **> 1.08** | **诱上，真实看好下盘** |

关键区分点：
- 真实定价：升盘后庄家在三个维度（盘口/水位/欧赔）同步收紧上盘赔付
- 浅盘升盘诱上：升盘后庄家在水位维度**扩大**上盘赔付敞口，欧赔即时回调暴露诱导意图

**规则 3：置信度分级**

| 满足要件数 | 置信度区间 | 说明 |
|-----------|-----------|------|
| 5 项全满 | 75~85 | 双重诱上确认，反向结论高置信度 |
| 4 项 | 65~75 | 诱上嫌疑较高，反向结论中高置信度 |
| 3 项 | 55~65 | 诱上嫌疑中等，反向结论中等置信度 |
| ≤ 2 项 | 不触发 | 按标准规则处理 |

本案满足 5 项全满，正确置信度应为 **78/100**（看好主队浙江），而非 73/100（看好客队山东）。

**规则 4：比分预测修正**

当识别为浅盘升盘诱上组合时，比分预测必须包含**主队 2 球+ 净胜比分**，不可仅预测客队小胜：

| 错误预测（本案） | 正确预测 |
|---------------|---------|
| 1:2、0:1（客队小胜） | **2:0、3:1、2:1**（主队净胜 2 球+） |

原因：双重诱上组合暗示庄家对主队有较强信心，不仅看好主队赢球，且预判主队有净胜 2 球能力。

## 十四、浅盘凯利信号失真修正（客胜凯利 1.03~1.05 灰色地带）

### 教训来源
同上（浙江 4-1 山东泰山）

### 错误判断
初盘① Pinnacle 客胜凯利 1.042（> 1.04），套用 `feedback_preupgrade_kelly_priority.md`"升盘前凯利趋势优先"规则，判定为"升盘前已有真实信号，升盘是真实定价"。实际主队大胜，客胜凯利信号失效。

### 核心规则

**规则 1：浅盘场景下凯利信号的结构性失真**

当初盘开小（实际盘口 < 理论盘口 0.25 球+）时，客胜凯利 > 1.0 可能是**盘口浅开导致的数学结果**，而非庄家真实看好客队：

| 盘口性质 | 客胜赔率特征 | 客胜凯利表现 | 失真原因 |
|---------|------------|-------------|---------|
| 初盘合适/开大 | 正常定价 | < 1.0 = 看好客队 | 无失真 |
| **初盘开小** | **结构性偏高** | **> 1.0 可能是数学结果** | **浅开 → 客队让球负担轻 → 客胜赔率相对市场偏高** |

**规则 2：凯利强信号阈值的浅盘修正**

| 初盘性质 | 客胜凯利强信号阈值 | 灰色地带 | 说明 |
|---------|------------------|---------|------|
| 初盘合适/开大 | < 0.96（看好）、> 1.05（不看好） | 0.96~1.05 | 标准阈值 |
| **初盘开小** | **< 0.95（看好）、> 1.08（不看好）** | **0.95~1.08** | **阈值放宽，灰色地带扩大** |

本案：初盘开小 + 客胜凯利 1.042 → 落入浅盘灰色地带（0.95~1.08），**不构成"升盘前真实信号"**。

**规则 3：灰色地带凯利的正确处理**

当客胜凯利落入灰色地带时，**不可单独作为定性依据**，必须结合以下信号综合判断：

| 验证项 | 真实看好客队 | 凯利信号失效 |
|--------|------------|-------------|
| 升盘后主水方向 | 下降或维持（≤ 5 点） | **反升 ≥ 15 点** |
| 即时欧赔延续性 | 延续下降或稳定 | **回调 ≥ 0.05** |
| 升盘后主凯利 | < 1.05 | **> 1.08** |

本案：升盘后主水反升 25 点 + 即时欧赔回调 0.10 + 主凯利 > 1.08 → **客胜凯利 1.042 信号失效，反向看好主队**。

**规则 4：与"升盘前凯利趋势优先"规则的关系**

`feedback_preupgrade_kelly_priority.md` 规则有边界条件：

| 场景 | 升盘前凯利趋势优先 | 需重新验证 |
|------|------------------|-----------|
| 初盘合适/开大 + 升盘前凯利 < 0.97 | **适用**（升盘前真实信号） | — |
| **初盘开小 + 升盘前凯利 1.03~1.05** | — | **不适用，必须验证升盘后水位/欧赔/主凯利** |

## 十五、中超主场反弹潜力评估（主场交锋能力 + 客队客场大败前科）

### 教训来源
同上（浙江 4-1 山东泰山）

### 错误判断
过度放大山东泰山近期状态优势（近 5 场 4 胜 1 负）和积分差（6 分），严重低估浙江主场交锋能力（历史主场对山东 2 胜 2 平 1 负，不败率 80%）和山东客场大败前科（近 10 场客场 1-4 青岛、2-4 云南）。

### 核心规则

**规则 1：中超主场反弹潜力的四项触发条件**

当以下条件同时满足时，主场反弹潜力必须作为核心权重：

| 条件 | 阈值 | 本案数据 |
|------|------|---------|
| ① 主队历史主场交锋不败率 | ≥ 60%（近 5 场） | 80%（2 胜 2 平 1 负） |
| ② 客队近 10 场客场大败前科 | 存在净负 ≥ 3 球记录 | 1-4 青岛、2-4 云南 |
| ③ 主队近况输球对手质量高 | 输给联赛前 3 强队 | 0-4 成都（第 1） |
| ④ 积分差 | ≤ 10 分 | 6 分 |

**规则 2：权重分配修正**

| 因素 | 标准权重 | 触发四项条件后权重 | 本案应用 |
|------|---------|------------------|---------|
| 客队近期状态优势 | 25% | **10%** | 山东 4 胜 1 负被客场大败前科抵消 |
| 主队近况低迷 | 20% | **10%** | 浙江输球对手质量高（榜首成都） |
| 主队主场交锋能力 | 15% | **35%** | 浙江主场对山东不败率 80% |
| 中超主场反弹规律 | 10% | **25%** | 低迷期遇客场不稳定队 |
| 盘口信号 | 30% | 20% | 需结合基本面 |

**规则 3：近况对手质量量化（客队版扩展）**

不可简单统计"客场 N 战 X 胜"作为客队状态指标，必须按对手分级：

| 客场对手等级 | 输球的信号意义 |
|------------|--------------|
| 联赛前 4 强队（榜首集团） | **输给强队 ≠ 状态崩溃**（不可作为支持上盘的信号） |
| 同级别中游队 | 输给同级别对手 = 状态真实低迷 |
| 联赛后 4 弱队 | 输给弱队 = 客场严重崩盘（强信号） |

本案：浙江客场 3 负中有 0-4 成都（第 1）、0-3 大连（第 3）→ 输给榜首集团占主导，"客场极弱"判断不成立。

**规则 4：置信度上限**

中超主场反弹场景置信度上限 **80**（非 85+）：
- 若盘口信号与基本面反弹潜力一致 → 置信度 75~80
- 若盘口信号与基本面反弹潜力冲突 → 优先采信基本面，置信度 65~75

本案：浅盘升盘诱上组合（看好主队）+ 主场反弹潜力高 → 一致，正确置信度 **78/100**。

## 十六、升盘后回降的正确识别（非"全程不变盘"）

### 教训来源
Chelsea 2-1 Tottenham Hotspur（2026-05-20 英超第37轮）

### 错误判断
初盘① -0.25 → 初盘②升至 -0.5 → 临场②降回 -0.25，机械套用第十一章规则 1（全程不变盘+给水+临场协同压主胜=诱上变体），推荐下盘（热刺）。实际切尔西 2:1 主场胜，**上盘赢球赢盘**，预测方向完全错误。

### 核心规则

**规则 1：升盘后回降 ≠ "全程不变盘"，应走定价修正路径**

第十一章规则 1（诱上型变体）的前提是"全程 ≥ 8 节点盘口不变盘"。当盘口经历**升盘→回降**时，不属于"不变盘"场景：

| 变盘路径 | 性质 | 正确匹配 |
|---------|------|---------|
| 全程不变盘（如 10 节点均为 -0.5） | 盘口锁定 | 可触发诱上型变体识别 |
| **升盘后回降**（如 -0.25→-0.5→-0.25） | **定价修正** | **军规 #3（修正→转场景三）或军规 #10 变体** |
| 连续降盘（如 -0.75→-0.5→-0.25） | 定价修正 | 军规 #3 |

**规则 2：升盘后回降 + 主水先降后反升 = 临场诱下组合**

当变盘路径满足以下特征时，识别为"临场诱下"（与"诱上变体"镜像）：

| 阶段 | 操作 | 散户心理 |
|------|------|----------|
| 初盘②~中盘 | 升盘（如 -0.25→-0.5）+ 给水（主水 2.07→1.93） | "庄家看好上盘 → 押上盘" |
| 临场② | 降回初盘水平（-0.5→-0.25）+ 主水压低（1.93→1.85） | "降盘了 → 上盘更稳了" |
| 临场④ | **主水反升**（1.85→1.96）+ **欧赔主胜跳升**（2.16→2.28） | "主水升高 → 下盘更划算" |

判定要件（全部满足即视为临场诱下）：
- 初盘②~中盘升盘 ≥ 0.25 球
- 临场②降回初盘水平或更低
- 临场②主水压低（降幅 ≥ 10 点）
- **临场④主水反升 ≥ 8 点 + 欧赔主胜跳升 ≥ 0.10**

**规则 3：降盘 + 主水先降后反升的双重验证**

参考第七章规则 1（降盘+主水反升 ≠ 军规 #10），本案是其变体：

| 场景 | 主水变化 | 判定 |
|------|---------|------|
| 降盘 + 主水持续下降或维持 | 临场②③④主水 1.85→1.83→1.80 | 军规 #10，真实看好上盘 |
| **降盘 + 主水先降后反升** | **临场②主水 1.85 → 临场④主水 1.96** | **诱下变体，封盘前诱导下盘资金** |

关键区分：主水是否在临场④（封盘前 ≤ 30 分钟）出现反升 ≥ 8 点。

**规则 4：H2H 主场碾压优势在本赛季排名同向时的权重上调**

当满足以下条件时，H2H 主场优势必须作为核心权重（35%）：

| 条件 | 阈值 | 本案数据 |
|------|------|---------|
| ① H2H 主场不败率 | ≥ 80%（近 5~10 场） | 100%（5胜1平0负） |
| ② 本赛季积分差 | 主队领先 ≥ 8 分 | 11 分（49 vs 38） |
| ③ H2H 近期延续性 | 近 5 场主队胜率 ≥ 60% | 80%（4胜1平） |

**与既有规则的关系**：
- [[feedback_h2h_vs_current_season]]（H2H 与本赛季排名反向）规定：H2H 占优方本赛季排名反落后时，本赛季权重 > H2H
- **本章为其补充**：H2H 占优方本赛季排名**同向占优**时，H2H 权重应上调至 35%（非标准 15%）

**规则 5：浅盘场景下亚盘主凯利 < 0.90 的灰色地带**

参考第十四章规则 1（浅盘场景下客胜凯利 1.03~1.05 是灰色地带），镜像规则：

| 初盘性质 | 亚盘主凯利强信号阈值 | 灰色地带 |
|---------|------------------|---------|
| 初盘合适/开大 | < 0.90（看好上盘） | 0.90~0.95 |
| **初盘开小或浅盘（≤ -0.25）** | **< 0.85（看好上盘）** | **0.85~0.95** |

本案：初盘 -0.25（浅盘）+ Pinnacle 亚盘主凯利 0.858~0.886 → 落入浅盘灰色地带，**不构成"主队被压制"的强信号**，必须验证临场④水位方向。

## 十七、主场近期崩溃 vs H2H 碾压优势的权重平衡

### 教训来源
同上（Chelsea 2-1 Tottenham Hotspur）

### 错误判断
过度放大 Chelsea 主场近 3 场全败（0-3 诺丁汉、0-1 曼联、0-3 曼城）权重至 40%，严重低估 H2H 主场碾压优势（5胜1平0负，不败率 100%）权重至 15%。

### 核心规则

**规则 1：主场近期崩溃的对手质量量化**

不可简单统计"主场近 N 场战绩"作为状态指标，必须按对手质量分级：

| 主场对手等级 | 输球的信号意义 |
|------------|--------------|
| 联赛前 3 强队（榜首集团） | **输给强队 ≠ 主场能力崩溃**（不可作为支持下盘的信号） |
| 同级别中游队 | 输给同级别对手 = 主场能力真实低迷 |
| 联赛后 4 弱队 | 输给弱队 = 主场严重崩盘（强信号） |

本案：Chelsea 主场 3 负中有 0-3 曼城（第 2）、0-1 曼联（第 3）、0-3 诺丁汉（当时状态火热，近 5 场 4 胜）→ **输给强队占主导**，"主场崩溃"判断应大幅弱化。

**规则 2：H2H 主场碾压 + 本赛季排名同向的权重分配**

| 因素 | 标准权重 | H2H 主场不败率 ≥ 80% + 本赛季积分差 ≥ 8 分时权重 |
|------|---------|------------------------------------------|
| H2H 主场优势 | 15% | **35%** |
| 主队近况低迷 | 25% | **15%**（需量化对手质量） |
| 客队近况优势 | 20% | **10%**（需验证客场稳定性） |
| 本赛季积分差 | 15% | **25%** |
| 盘口信号 | 25% | 15% |

**规则 3：客队"客场表现优于主场"的陷阱**

当客队近期"客场战绩优于主场战绩"时，不可直接判定为"客场战斗力强"，必须验证：

| 验证项 | 真实客场强 | 虚假信号 |
|--------|----------|---------|
| 客场对手质量 | 客场赢同级别或强队 | **客场赢弱队，输强队** |
| 客场稳定性 | 客场连续 3 场+ 不败 | **客场胜负交替，不稳定** |
| 客场净胜球 | 客场平均净胜球 > 0 | **客场小胜小负，净胜球接近 0** |

本案：Tottenham 客场近 3 场 2 胜 1 负（1-2 维拉、0-1 狼队、1-0 桑德兰），胜负交替 + 小比分 → **客场稳定性不足**，不构成"客场战斗力强"的强信号。

**规则 4：英超主场交锋碾压优势的特殊性**

英超主场优势显著高于其他联赛，当 H2H 主场不败率 ≥ 80% 时：

| 联赛 | H2H 主场不败率 ≥ 80% 权重 | 说明 |
|------|------------------------|------|
| 中超 | 25% | 主场优势不稳定 |
| 五大联赛（除英超） | 30% | 主场优势中等 |
| **英超** | **35%** | **主场优势最显著** |

原因：英超主场氛围、草皮质量、裁判主场哨、球迷压力均为五大联赛最强。

## 十八、强队深盘 + 高平赔的世界杯/杯赛冷平风险

### 教训来源

England 0-0 Ghana（2026-06-24 世界杯小组赛）。详见 [[strong_team_deep_handicap_cup_cold_draw]]。

### 错误判断

欧赔主胜从 1.22~1.23 压至 1.18~1.20、亚盘从 England -1.75 升至 -2.0 且上盘水位由高水（2.10+）压至均衡，判为"升盘降赔的真实强化"，主推上盘 England + 主胜，比分 3-0 / 3-1，置信度 **76**。实际 **0-0 平局**，深盘上盘全输，主胜未出，总进球 0（2.5/3.0/3.5 全小球赢）。

### 核心规则

**规则 1：强队"初盘合适 + 升盘至深盘"——#11 诱上表象不可过滤。** 军规 #11 的触发不以"升盘后主水大幅反升"为必要条件。强队深盘场景中，即使升盘后上盘水位不升反降（看似真实强化），升盘动作本身已把热度推满、完成诱上（降赔式诱上）。不可用"主水未反升 → 不是诱上"一句话过滤 #11。

**规则 2：世界杯/杯赛强弱盘，平赔 ≥ 7.0 ≠ 排除平局。** 联赛里高平赔常意味平局不利，但杯赛强弱对话中弱队龟缩 + 强队松懈 → 高赔冷平为真实可兑现尾部事件。平赔越高，冷平一旦兑现破坏性越大，必须保留独立的小概率高权重，不可归零。

**规则 3：深盘主盘口双方凯利接近 1.0 = 市场对深盘"无保护"，非"支持深盘"。** -2.0 主盘口双方凯利都接近 1.0，应理解为市场对这条深盘没有明确方向保护，分界点恰在最易翻车处，风险须上修到"主胜本身都可能不稳"，而非只停在"赢 2 球走盘"。

**规则 4：2.5 大球低水 + 3.5 小球低水并存 → 比分必须收敛。** 两者同时存在时正确解读是"可能有球但不支持极端打穿"，不可只取 2.5 大球低水推高进球（那常是配合深让盘制造"强队大胜"叙事）。比分候选必须保留 0-0、1-0、1-1 等低比分，禁止只给 3-0 / 3-1。

**规则 5：深盘 + 杯赛 + 缺终指 + B/C 凯利分歧，任意 ≥ 2 项 → 置信度封顶 ≤ 65~70。** 本案四项全中却给 76，严重低估反向风险。个别公司平局凯利极低（10Bet ~0.886、Dafabet ~0.743）与主流公司平局凯利 >1.03 并存时，应判 B/C 冲突而非"基本一致"，置信度须下修至 60~65。

### 与既有规则的关系

| 规则 | 关系 |
|------|------|
| 第一章（欧亚同步升盘=真实定价） | **本章为强队深盘场景的边界条件**：升盘降赔在深盘+杯赛下可能是诱上而非真实定价 |
| 第十章规则 2（升盘+主水反升=升盘版诱上） | 本章规则 1 为其补充：强队深盘"升盘+主水反降"亦可为诱上 |
| 第八章（H2H 高平局率强制修正） | 本章规则 2 为其杯赛扩展：强弱深盘的高平赔冷平须独立保权重 |
| reference_over_under.md 第四章（大小球与亚盘深度联动） | 本章规则 4 为其细化：大球低水+高线小球低水并存须收敛比分 |
