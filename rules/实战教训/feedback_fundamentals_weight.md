# 基本面与盘口权重分配

> **触发条件**：基本面与盘口方向矛盾；基本面一边倒（疑似碾压级）；涉及战意/动力差异（保级、争冠、杯赛、德比）。

## A. 权重分配基准表

| 场景 | 基本面权重 | 盘口信号权重 | 说明 |
|------|-----------|-------------|------|
| 基本面与盘口方向一致 | 40% | 60% | 正常分配 |
| 基本面与盘口方向矛盾（盘口加深） | 20% | 80% | 庄家可能有非公开信息 |
| 杯赛 + 基本面矛盾 | 15% | 85% | 杯赛状态独立于联赛 |
| 基本面碾压 + 盘口浅开 | 30% | 70% | 参考 B 节碾压级规则 |

**杯赛特殊性**：杯赛（尤其淘汰赛/半决赛）状态与联赛可以完全脱节；联赛连败 ≠ 杯赛连败（动力、轮换、战术不同）；主场杯赛半决赛/决赛存在「杯赛 DNA」加成，历史交锋权重应上调。

## B. 碾压级基本面判定与诱上信号过滤

> 来源：AS Roma 2-0 Lazio（2026-05-17 意甲罗马德比）。临场④主水反升 + 即时回调，机械套用诱上变体推荐下盘，实际上盘赢球赢盘。

**规则 B1：碾压级基本面五要素（满足 ≥ 3 项成立）**：

| 基本面要素 | 阈值 |
|-----------|------|
| 联赛积分差 | **≥ 15 分** |
| 客队近 3 场战绩 | **2 场+ 大败（净负 ≥ 2 球）** |
| 客队赛程压力 | **3~7 天内有 2 场+ 重要比赛（杯赛+联赛连战）** |
| 主队主场近况 | **近 3 个主场全胜 + 进 5 球+** |
| 历史交锋 | **近 5 场主队胜率 ≥ 60%（不含走盘）** |

碾压级成立时，临场④主水反升 ≤ 10 点应视为**风控收紧**（散户大量押上盘时庄家平衡赔付敞口），不可触发诱上识别。

**规则 B2：诱上变体识别的前置条件 = 先排除碾压级基本面**。判定优先级（自上而下）：

1. **基本面碾压** → 风控收紧 → 上盘真实看好
2. 基本面均衡 + 全程不变盘 + 给水 + 临场④协同压主胜 → 诱上变体（[feedback_lure_variants.md](feedback_lure_variants.md) B 节）
3. 基本面均衡 + 临场④升盘 + 主水反升 → 升盘版诱上（[feedback_sync_pricing.md](feedback_sync_pricing.md) B 节）

**规则 B3：临场④主水反升幅度分级（不可"任何反升均视为诱上"）**：

| 反升幅度 | 性质 | 处理 |
|---------|------|------|
| ≤ 5 点 | 正常市场波动 | 忽略 |
| **6~10 点** | 风控调整或诱上 | **碾压级基本面 → 风控；均衡基本面 → 诱上嫌疑** |
| 11~15 点 | 诱上嫌疑较高 | 即使有强基本面，置信度扣减 5 分 |
| ≥ 16 点 | 诱上确认 | 不论基本面，警惕诱上 |

**规则 B4：欧赔即时回调幅度分级**：

| 欧赔即时回调 | 碾压级基本面 | 均衡基本面 |
|---------------|-----------|-----------|
| 0.01~0.03 | **正常市场调整，忽略** | 诱上嫌疑信号 |
| 0.04~0.07 | 中性观察，结合主水方向 | 诱上确认信号 |
| ≥ 0.08 | 即使有强基本面也警惕 | 诱上强确认 |

## C. 客队疲态系数与德比修正

**规则 C1：疲态系数（作为支持主场上盘的核心权重）**：

| 疲态信号 | 权重影响 |
|---------|---------|
| 距上场重要比赛 ≤ 3 天 | 体能扣减客队战力 15% |
| 上场重要比赛失利（如杯决败北） | 心理扣减客队战力 10% |
| 近 2 场失球 ≥ 5 个（防线崩塌） | 防守效率扣减 20% |
| 客场连战 | 长途+客场环境扣减 10% |

触发多项时叠加（如 Lazio 三项 ≈ -35%，「残血出战」）；此时即使盘口有轻微诱上信号，优先采信基本面碾压判断。

**规则 C2：同城德比「小球+1-0」惯性 ≠ 走盘风险**：

| 场景 | 处理 |
|------|------|
| 德比 + 双方实力接近 + 同积分区 | 走盘风险高，扣上盘置信度 5~10 分 |
| **德比 + 实力碾压（积分差 ≥ 15 分）+ 客队疲态** | **走盘惯性被打破，按碾压逻辑预判 2 球+ 比分** |

## D. 战意与动力

**规则 D1：保级队主场动力权重上调**（来源：Newcastle 3-1 Brighton）。主队处于保级区（排名 15-18）且主场作战时：近况权重下调至 15%，保级战意 + 主场优势上调至 25%，盘口信号维持 60%。

**规则 D2：冠军压力下客场作战 = 保平争胜，而非「必须赢球」**（来源：Bournemouth 1-1 Man City，2026-05-20 英超冠军争夺战）。客场全力进攻风险高（可能 0 分），保平争胜风险低（至少 1 分），冠军压力下理性选择是**降低风险**。不可将「必须拿分」等同「必须赢球」而推客胜。

**规则 D3：碾压级必须满足「对手崩溃」条件**（来源：同上）。历史交锋碾压 + 战略压力仅是碾压级的 2 项条件；对手近况良好（如 6 场不败）= **未崩溃**，不构成碾压级基本面，不可提升置信度。

**规则 D4：保级死战 vs 无欲无求 = 极端动力差异**，关闭凯利双重极端强制反转，动力权重升至 55% → 详见 [feedback_csl_fundamentals.md](feedback_csl_fundamentals.md) E 节。

## 本主题检查项

- [ ] 基本面与盘口矛盾时，是否已下调基本面权重（20/80）？
- [ ] 杯赛是否已独立于联赛状态评估？
- [ ] 碾压级判定是否逐项核对五要素并 ≥ 3 项？对手未崩溃（近况不败）不可算碾压
- [ ] 诱上变体识别前是否已先排除碾压级基本面？
- [ ] 临场④主水反升是否按分级判定（≤5 忽略 / 6~10 看基本面 / 11~15 扣 5 分 / ≥16 诱上确认）？
- [ ] 欧赔即时回调 ≤ 0.03 在碾压级下是否识别为正常调整？
- [ ] 客队疲态信号（≤3 天重要比赛 / 上场失利 / 近 2 场失 5 球）是否已折算战力扣减？
- [ ] 德比是否实力碾压（≥15 分 + 客队疲态）？若是，不套用小比分走盘扣减
- [ ] 保级区主队主场：保级战意权重是否上调、近况权重是否下调？
- [ ] 冠军争夺战客场方：是否按「保平争胜」而非「必须赢球」定性？
