# 强队深盘 + 高平赔，世界杯/杯赛冷平风险

## 错误案例

**England 0-0 Ghana**（2026-06-24 世界杯小组赛）

- **错误判断**：欧赔主胜从 1.22~1.23 压至 1.18~1.20、亚盘从 England -1.75 升至 -2.0 且 England 水位由高水压至均衡，判为「升盘降赔的真实强化」，主推上盘 England + 主胜，比分 3-0 / 3-1，置信度 **76**。
- **实际结果**：**0-0 平局**，England -1.75 / -2.0 上盘全输，主胜未出，总进球 0（2.5/3.0/3.5 全部小球赢）。
- **根本错误**：把「初盘合适 + 升盘至深盘」的 #11 诱上表象当作真实强化过滤掉；把强队主胜的自然热度误读为胜场必出；高平赔误读为「平局必死」；深盘场景置信度给到 76 明显过高。

---

## 核心规则

### 规则 1：强队「初盘合适 + 升盘至深盘」——#11 诱上表象不可过滤

军规 #11（初盘合适 + 升盘 = 上盘热、诱上、下盘胜）的触发，**不以「升盘后主水大幅反升」为必要条件**。在强队深盘场景中，即使升盘后上盘水位不升反降（看似真实强化），升盘动作本身已经把热度推高、完成诱上。

- 不可用「主水未大幅反升 → 不是诱上」一句话过滤 #11。
- England 由 -1.75 升至 -2.0、上盘水位由 2.10+ 压到均衡，正是这种「降赔式诱上」。

### 规则 2：世界杯/杯赛强弱盘，高平赔 ≠ 排除平局

平赔被抬到 **7.0+** 在联赛里常意味"平局不利"，但在世界杯/杯赛强弱对话中，弱队龟缩摆大巴 + 强队心理松懈 → **高赔冷平**是真实可兑现的尾部事件。

- 平赔越高，单注赔付越大，一旦兑现即为高破坏性冷门。
- 强弱深盘场景必须为「高平赔冷平」保留独立的小概率高权重，不可归零。

### 规则 3：深盘主盘口双方凯利接近 1.0 = 市场对深盘"无保护"，非"支持深盘"

-2.0 主盘口双方凯利都接近 1.0，不应理解为「市场认可 England 赢 2 球」，而应理解为 **市场对这条深盘没有明确方向保护**——分界点恰在最易翻车处，风险应向上修到"主胜本身都可能不稳"，而非只停留在"赢 2 球走盘"。

### 规则 4：2.5 大球低水 + 3.5 小球低水并存 → 比分必须收敛

当 2.5 大球低水与 3.5 小球低水**同时存在**，正确解读是「可能有球，但不支持极端打穿」，而非「3 球中心」。

- 不可只取 2.5 大球低水推高进球预期（那往往是配合深让盘制造"强队大胜"叙事）。
- 比分候选必须保留 1-0、0-0、1-1 等低比分，禁止只给 3-0 / 3-1。

### 规则 5：深盘 + 杯赛 + 缺终指 + 冷平风险并存 → 置信度封顶 ≤ 65~70

满足以下任意 ≥ 2 项时，置信度主动封顶，不得超过 70（理想 ≤ 65）：

| 条件 | 本案 |
|------|------|
| 让球达 -1.75 / -2.0 深盘 | ✓ |
| 世界杯 / 杯赛（强弱对话冷平风险高） | ✓ |
| 无终指 / 基本面薄弱 | ✓（盲推无基本面） |
| B/C 凯利存在分歧 | ✓ |

### 规则 6：部分公司平局低凯利 = B/C 冲突，必须降置信度

主流公司平局凯利 >1.03（看似排除平局）与个别公司平局凯利极低（如 10Bet ~0.886、Dafabet ~0.743）并存时，应判定为 **B/C 冲突**，而非"基本一致"。凯利分歧场景置信度必须下修，本案应标冲突、降至 60~65。

---

## Why（为什么这条规则重要）

1. 强队深盘的"升盘降赔"极易被误读为真实强化，实则是降赔式诱上——上盘热度推满后死亡。
2. 世界杯/杯赛强弱盘的冷平不是噪音，是结构性尾部风险，高平赔正是其赔付特征。
3. 深盘主盘口凯利均衡是"无保护"信号，常被反向当作"市场认可深盘"。
4. 大小球双低水（大球低 + 高线小球低）是"有球不打穿"的收敛信号，单取一侧必偏。
5. 深盘 + 缺基本面 + 杯赛冷平风险叠加时，76 分这类高置信度严重低估了反向风险。

---

## How to apply（如何应用）

### 分析时

1. 见「强队初盘合适 + 升盘至 -1.75/-2.0」，先按 #11 诱上预警处理，不得以"主水未反升"过滤。
2. 平赔被抬至 7.0+ 时，在杯赛/强弱盘场景保留「高平赔冷平」权重，不写"排除平局"。
3. 深盘主盘口双方凯利接近 1.0 → 标为"市场对深盘无保护"，风险上修到主胜可能不稳。
4. 2.5 大球低水 + 3.5 小球低水并存 → 比分收敛，保留低比分（0-0/1-0/1-1）。
5. B/C 凯利分歧 → 判冲突、降置信度。
6. 深盘 + 杯赛 + 缺终指任意 ≥2 项 → 置信度封顶 ≤ 70（理想 ≤65）。
