# 强队深盘 / 杯赛冷平

> **触发条件**：强队让 -1.5 及以上深盘；世界杯/洲际杯/杯赛强弱对话；热门方欧赔被压至 1.2x 以下且平赔 ≥ 7.0。
> A/B 为姊妹场景：同源于强队升盘诱上——上盘死，一走向冷平（A），一走向受让方直接反胜（B）。
> 国家队赛事基本面读法配套 `reference_national_team.md` 第四、六章。

## A. 强队深盘 + 高平赔的冷平风险

> 来源：England 0-0 Ghana（2026-06-24 世界杯小组赛）。欧赔压至 1.18~1.20 + 亚盘 -1.75 升 -2.0 且上盘水位压至均衡，判"升盘降赔真实强化"推上盘+主胜 3-0/3-1（置信度 76），实际 0-0 冷平，深盘全输。

**规则 A1：强队「初盘合适 + 升盘至深盘」= 降赔式诱上，#11 表象不可过滤**。军规 #11 的触发**不以**"升盘后主水大幅反升"为必要条件——强队深盘场景中即使升盘后上盘水位不升反降（看似真实强化），升盘动作本身已把热度推满、完成诱上。不可用"主水未反升 → 不是诱上"过滤。

**规则 A2：杯赛强弱盘，平赔 ≥ 7.0 ≠ 排除平局**。联赛高平赔常意味平局不利，但杯赛强弱对话中弱队龟缩 + 强队松懈 → 高赔冷平是真实可兑现尾部事件。平赔越高冷平破坏性越大，必须保留独立的小概率高权重，不可归零。

**规则 A3：深盘主盘口（-1.75/-2.0）双方凯利接近 1.0 = 市场对深盘「无保护」**，非"支持深盘"。分界点恰在最易翻车处，风险须上修到"主胜本身都可能不稳"，而非只停在"赢 2 球走盘"。

**规则 A4：2.5 大球低水 + 3.5 小球低水并存 → 比分必须收敛**。正确解读是"可能有球但不支持极端打穿"，不可只取 2.5 大球低水推高进球（那常是配合深让盘制造"强队大胜"叙事）。比分候选必须保留 0-0、1-0、1-1，禁止只给 3-0/3-1。

**规则 A5：置信度封顶**。深盘 / 杯赛 / 缺终指（基本面薄弱）/ B/C 凯利分歧，任意 ≥ 2 项 → 置信度 ≤ 65~70。个别公司平局凯利极低（如 <0.90）与主流平局凯利 >1.03 并存 = **B/C 冲突**（非"基本一致"），置信度下修至 60~65。

## B. 升盘至深盘后临场降温 = 受让方可能直接反胜

> 来源：Ecuador 2-1 Germany（2026-06-26 世界杯小组赛）。正确判出"Germany 穿 -1 不稳、亚盘看 Ecuador +1"（命中），但胜平负仍首选客胜 Germany → 亚盘对、胜平负全错。

**规则 B1：亚盘方向判对 ≠ 胜平负判对**。判定"热门穿深盘不稳、改看受让方 +1"时，必须同步把**受让方不败乃至受让方取胜**纳入胜平负备选，而非默认"热门小负赢盘 / 平局"。杯赛环境下受让 +1 的赢盘常由受让方直接赢球兑现。

**规则 B2：完整信号链 = 热门胜场动摇**（权重上调，非仅"不穿盘"）：
初盘合适 → 中盘剧烈升盘 → 临场上盘高水不降 → 热门胜赔回升 → 受让方赔率回压。
出现完整链条时把冷门（受让方不败/获胜）抬到有效备选权重，不可仅列风险提示。

**规则 B3：中盘剧烈欧亚同步加深，临场未延续 = 「制造强势题材」的风控盘**，非胜场确认。热门欧赔大幅压低 + 亚盘大幅加深，若临场不再延续（高水不降、赔率回升），须二次定性。

**规则 B4：热门胜赔从低点回升 = 「胜场降温」**（强于"仅穿盘降温"）。热门被压至低点（如 1.52）后临场回升（1.58~1.66），配合上盘高水 + 受让方赔率回压三者叠加 = 热门崩盘尾部预警。

**规则 B5：亚盘选受让方 + 热门胜赔明显回升时，比分候选必须含受让方取胜比分**（受让方 2:1 / 1:0），不可只给"热门小胜"。

## 本主题检查项

- [ ] 强队「初盘合适 + 升至 -1.75/-2.0」？→ 按军规 #11 诱上预警，不可用"主水未反升"过滤
- [ ] 杯赛强弱盘平赔 ≥ 7.0？→ 为高平赔冷平保留独立权重（不可写"排除平局"）
- [ ] 深盘主盘口双方凯利 ≈ 1.0？→ 标注"市场无保护"，风险上修到主胜不稳
- [ ] 2.5 大球低水与 3.5 小球低水并存？→ 比分收敛，保留 0-0/1-0/1-1
- [ ] 个别公司平局凯利极低 + 主流 >1.03 并存？→ 判 B/C 冲突并降置信度
- [ ] 深盘/杯赛/缺终指/B-C 分歧 ≥ 2 项？→ 置信度封顶 ≤ 65~70
- [ ] 判"热门穿盘不稳、看受让方 +1"时，受让方不败/取胜是否已进胜平负备选？
- [ ] 出现"升盘→临场降温"完整链条？→ 按"热门胜场动摇"定性（非仅不穿盘）
- [ ] 亚盘选受让方 + 热门胜赔回升？→ 比分候选含受让方 2:1 / 1:0
