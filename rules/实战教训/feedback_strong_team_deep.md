# 强队深盘 / 杯赛冷平（速查卡）

> 本卡合并自：`strong_team_deep_handicap_cup_cold_draw`（深盘高平赔冷平）、`upgrade_then_cooldown_underdog_can_win`（升盘降温受让方反胜）。二者为姊妹场景：同源于强队升盘诱上，上盘死一走向平局、一走向受让方反胜。
> **完整论证见** [reference_case_lessons.md](reference_case_lessons.md) **第十八章、第十九章**。

## A. 强队深盘 + 高平赔冷平（→ 十八章）

- **升盘诱上不可过滤**：强队"初盘合适 + 升盘至 -1.75/-2.0"即使升盘后主水不升反降，也已完成降赔式诱上，不可用"主水未反升→不是诱上"过滤军规 #11。
- **高平赔 ≠ 排除平局**：世界杯/杯赛强弱盘平赔 ≥7.0 时，弱队龟缩+强队松懈的高赔冷平是真实尾部事件，须保留独立小概率高权重。
- **深盘双方凯利≈1.0 = 无保护**：风险上修到"主胜本身可能不稳"，非只"赢 2 球走盘"。
- **2.5 大球低水 + 3.5 小球低水并存 → 比分收敛**：保留 0-0/1-0/1-1，禁止只给 3-0/3-1。
- **置信度封顶**：深盘 / 杯赛 / 缺终指 / B/C 凯利分歧，任意 ≥2 项 → ≤65~70。个别公司平局凯利极低 + 主流 >1.03 = B/C 冲突（非"基本一致"）。

## B. 升盘至深盘后临场降温 = 受让方可能直接反胜（→ 十九章）

- **亚盘对 ≠ 胜平负对**：判"热门穿深盘不稳、看受让方 +1"时，必须把**受让方不败乃至受让方取胜**纳入胜平负备选，而非默认"热门小负赢盘/平局"。
- **复合信号链**：初盘合适 → 中盘剧烈升盘 → 临场上盘高水不降 → 热门胜赔回升 → 受让方赔率回压 = 热门胜场动摇（非仅"不穿盘"）。
- **中盘剧烈加深若临场未延续** = 制造强势题材/风控盘，非胜场确认。
- **热门胜赔从低点回升** = 胜场降温（强于仅穿盘降温），热门崩盘尾部预警。
- **比分候选**须含受让方取胜比分（受让方 2:1 / 1:0）。
