# 客队"客场优于主场"的稳定性验证

## 核心规则

**客队"客场表现优于主场"时，不可直接判定为"客场战斗力强"**

必须验证以下三项：

| 验证项 | 真实客场强 | 虚假信号 |
|--------|----------|---------|
| 客场对手质量 | 客场赢同级别或强队 | **客场赢弱队，输强队** |
| 客场稳定性 | 客场连续 3 场+ 不败 | **客场胜负交替，不稳定** |
| 客场净胜球 | 客场平均净胜球 > 0 | **客场小胜小负，净胜球接近 0** |

## 权重调整

| 客场表现特征 | "客场战斗力强"权重 | 说明 |
|------------|----------------|------|
| 连续 3 场+ 不败 + 赢同级别队 + 净胜球 > 0 | **25%** | 真实客场强 |
| 胜负交替 + 小比分 + 净胜球接近 0 | **10%** | 客场稳定性不足 |
| 主要赢弱队 + 输强队 | **5%** | 虚假客场优势 |

## Why

"客场战绩优于主场战绩"可能是以下原因：
1. **真实客场强**：客场连续不败，赢同级别或强队，净胜球为正
2. **主场太差**：主场崩盘导致客场相对显得好，但客场本身并不强
3. **对手质量差异**：客场遇到的对手整体较弱，主场遇到的对手较强
4. **小样本波动**：近期客场恰好赢了几场，但胜负交替，不稳定

只有第 1 种情况才能支持"客场战斗力强"的判断。

## How to apply

1. 统计客队近 5~10 场客场战绩
2. 检查客场是否连续 3 场+ 不败（若胜负交替 → 稳定性不足）
3. 查询客场对手的联赛排名（若主要赢弱队 → 虚假优势）
4. 计算客场平均净胜球（若接近 0 → 小胜小负，不稳定）
5. 若客场胜负交替 + 小比分 + 净胜球接近 0，"客场战斗力强"权重下调至 10%
