# 欧亚同步升降盘定价研判

> **触发条件**：任一阶段出现盘口升降（升盘 / 降盘 / 先升后降）。
> 本文件先定性「真实定价 or 定价修正 or 诱导」；变体排查再走
> [feedback_lure_variants.md](feedback_lure_variants.md)（诱盘形态）、[feedback_late_stage_shift.md](feedback_late_stage_shift.md)（临场末端异动）。

## A. 欧亚同步升盘 = 真实定价（非诱盘）

> 来源：Chelsea 1-0 Leeds（2026-04-26 足总杯半决赛）。将升盘（平半→半球）判定为"诱上"推荐下盘，实际上盘 1-0 赢球赢盘。

**规则 A1：欧亚同步升盘 = 真实定价**。以下条件同时满足时，升盘必须视为庄家真实看好上盘：

- 欧赔主胜方向持续下降（如 2.12 → 2.05）
- 亚盘同步升盘（如平半 → 半球）
- 两个方向一致且在**早中期完成**（非临场突变）

此时不可套用军规 #4（开大+不变盘+诱上）或 #5（升盘诱上），升盘本身是定价修正。

**规则 A2：升盘前凯利极低信号不可被升盘后数据覆盖（底色信号）**

- 升盘前主凯利持续 < 0.97（尤其 < 0.90）= 庄家初盘阶段就真实看好主队
- 升盘后凯利回升至 1.0+ 是盘口加深后的数学结果，不代表庄家态度反转
- 正确做法：以升盘前的凯利趋势为「底色」，升盘后的凯利变化仅作微调参考

**规则 A3：反常识升盘 = 最强信号**。基本面明显不利（连败、状态崩溃）但庄家仍升盘时：庄家掌握超越公开信息的判断；越与基本面矛盾的盘口变动越可能含内幕信息。此时基本面权重大幅下调、盘口信号权重上调。

## B. 真实定价的边界条件（防教条化）

> 来源：云南玉昆 1-2 浙江（2026-05-06 中超）。临场④升盘 + 欧赔同步下降，机械套用规则 A 推荐上盘，实际上盘输盘。

**规则 B1：判定真实定价必须同时通过四项验证**：

| 验证项 | 真实定价 | 诱导操作 |
|--------|---------|---------|
| 升盘时机 | 早中期完成（中盘②或更早） | **临场③④（≤1.5h 内）才升盘** |
| 升盘后欧赔 | 继续同方向或稳定 | **即时节点出现欧赔回调**（主胜回升/客胜回调） |
| 升盘后主水 | 适度调整（±5 点内）或下降 | **主水大幅反升 ≥ 10 点** |
| 升盘幅度 | ≤ 0.25 球 | ≥ 0.25 球（在临场④） |

**规则 B2：升盘 + 主水持续上升 = 升盘版「诱上」**（与 [feedback_late_stage_shift.md](feedback_late_stage_shift.md) A 节「降盘+主水反升」互为镜像）：

- 升盘后主水变化 ≤ 5 点 = 正常修正
- 升盘后主水上升 6~10 点 = 中性观察
- **升盘后主水持续上升 ≥ 10 点（含即时延续上升）= 诱上信号**

**规则 B3：临场④升盘必须独立验证，不可直接套用早中期规则**：

| 升盘时机 | 性质 | 处理规则 |
|---------|------|---------|
| 初盘②~中盘② | 定价基准修正 | 直接套用「欧亚同步升盘=真实定价」 |
| 中盘③~临场① | 消息消化修正 | 套用真实定价，置信度扣减 5 |
| 临场②~③ | 资金调整 | 必须验证主水方向 + 即时节点延续性 |
| **临场④（≤30min）** | **知情资金窗口** | **必须三重验证（欧赔延续 / 主水方向 / 客胜凯利）** |

> 凯利侧配套：Pinnacle 客胜凯利 < 0.96 的优先级反转见 [feedback_kelly_signals.md](feedback_kelly_signals.md) D 节；
> 中超主场优势权重下调见 [feedback_csl_fundamentals.md](feedback_csl_fundamentals.md) F 节。

## C. 连续降盘 = 定价修正（军规 #3 路径）

> 来源：浙江 2-1 深圳新鹏城（2026-05-02 中超）。初盘开大 + 临场连续降盘，误套军规 #4 推荐下盘，实际上盘赢球赢盘。

**规则 C1：降盘 = 降低上盘赢盘门槛 = 对上盘有利**。若庄家真的不看好上盘，应升盘或维持：

| 操作 | 含义 | 对上盘影响 |
|------|------|-----------|
| 降盘（-0.75→-0.50→-0.25） | 降低让球深度 | **有利**（更容易赢盘） |
| 升盘（-0.50→-0.75→-1.0） | 加深让球深度 | **不利**（更难赢盘） |
| 不变盘 | 维持现有让球 | 需结合水位判断 |

**规则 C2：初盘开大 + 连续降盘 → 军规 #3（修正→转场景三），不是军规 #4**：

| 场景 | 正确匹配 |
|------|---------|
| 初盘开大 + 不变盘 + 上盘大热 + 降水至低水 | 军规 #4（诱上→下盘胜） |
| 初盘开大 + 早中期/临场降盘至合适 | **军规 #3（修正→转场景三）** |
| 转场景三后 + 临场再降盘 | **军规 #10（减小上盘赔付→上盘赢球赢盘）** |

关键区分：军规 #4 的前提是「不变盘」，一旦发生降盘走军规 #3 路径。

**规则 C3：欧赔主胜上升 + 亚盘降盘 = 同步定价修正（「上盘小胜」信号）**。欧赔放开主胜赔付 + 亚盘降低让球门槛 = 庄家认为上盘小胜（1 球）。与规则 A（同步升盘=真实定价）互为镜像。

> 边界：临场③④剧烈降盘（欧主胜 > 0.50 或亚盘 ≥ 0.5 球）**不适用本节**，走 [feedback_late_stage_shift.md](feedback_late_stage_shift.md) B/C 节。

## 本主题检查项

- [ ] 欧亚同步升盘且早中期完成？→ 按真实定价处理，不套诱上军规（A1）
- [ ] 升盘前主凯利持续 < 0.97？→ 保留为底色信号，升盘后回升不代表反转（A2）
- [ ] 升盘发生在临场④？→ 不可直接套真实定价，执行 B3 三重验证
- [ ] 升盘后主水持续反升 ≥ 10 点（含即时）？→ 升盘版诱上（B2）
- [ ] 即时节点欧赔是否回调？→ 回调 = 信号方向反转（B1）
- [ ] 连续降盘是否误套军规 #4？（#4 前提是不变盘，降盘走 #3 路径）（C2）
- [ ] 欧主胜升 + 亚盘降发生在早中期？→ 上盘小胜；临场末端剧烈异动除外（C3）
