# 中超球队单场大败 ≠ 持续崩溃

## 核心规则

**中超球队状态波动大，单场大败不代表持续崩溃**

当以下条件满足时，**不可判定为"状态崩溃"**：
- 中超球队单场大败（如 0-5、0-4）
- 但大败前/后有大胜强队记录（如 4-1 胜前 5 名球队）
- 或大败对手为强队（前 3 名）

此时单场大败权重应降至 **15%**，近 3 场整体趋势权重上调至 **25%**。

## Why

**中超球队特点**：
- 状态波动大，单场大败后立即反弹常见
- 攻强守弱型球队常见（大胜大败交替）
- 主客场差异大，客场大败不代表主场崩溃

**大败后大胜强队 = 攻击力爆发**：
- 大败后立即大胜强队 = 攻击力强 + 防守不稳，非单向崩溃
- 大胜强队 = 对强队有爆发力，非"真实弱队"

## How to apply

### 验证清单

在判定"状态崩溃"前，必须逐项验证：

1. **大败后是否有大胜记录？**
   - 是（如 0-5 后 4-1 胜强队）→ 波动大，非崩溃 → 权重 15%
   - 否（如连续 3 场大败）→ 真实崩溃 → 权重 35%

2. **大败对手是否为强队？**
   - 是（前 3 名）→ 输强队，非崩溃 → 权重 15%
   - 否（中下游球队）→ 真实崩溃 → 权重 30%

3. **近 3 场是否连败？**
   - 是 → 真实崩溃 → 权重 35%
   - 否（胜负交替）→ 波动大，非崩溃 → 权重 15%

4. **大败场地是否与本场相反？**
   - 是（客场大败 + 本场主场）→ 主客场差异，非崩溃 → 权重 10%
   - 否（主场大败 + 本场主场）→ 主场崩溃 → 权重 30%

### 权重分配

| 场景 | 单场大败权重 | 近 3 场趋势权重 |
|------|------------|---------------|
| 大败后大胜强队 | **15%** | **25%** |
| 大败对手为强队 | **15%** | **25%** |
| 连续 3 场大败 | **35%** | **30%** |
| 客场大败 + 本场主场 | **10%** | **20%** |
| 主场大败 + 本场主场 | **30%** | **25%** |

### 置信度调整

- 单场大败 + 大败后大胜强队 → 崩溃权重 15%，置信度不扣分
- 单场大败 + 连续 3 场大败 → 崩溃权重 35%，置信度扣 15 分
- 单场大败 + 大败对手为强队 → 崩溃权重 15%，置信度不扣分

## 案例

**河南 0-2 浙江（2026-05-30 中超）**：
- 浙江 6 天前主场 0-5 惨败辽宁
- 但此前 4-1 胜山东泰山（5 名）
- 误判为"真实状态崩溃"，权重 30%
- 实际：大败后大胜强队 = 攻击力爆发 + 防守不稳，非单向崩溃，权重应为 15%
- 浙江 0-2 客胜河南

## 边界条件

**触发"真实崩溃"的场景**：
1. 连续 3 场大败（净负 ≥ 3 球）
2. 近 5 场输给 3+ 支中下游球队（排名 10 名以后）
3. 主场连续 2 场大败（净负 ≥ 3 球）
4. 大败后无任何反弹迹象（近 3 场 0 胜）
