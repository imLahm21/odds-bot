# 凯利指数分析参考手册

> 本文件供 LLM 分析赔率数据时作为规则库参考。
> 数据中的凯利指数字段由 main.py 自动计算：凯利 = 该公司赔率 / 市场平均赔率（简易凯利）。

---

## 一、凯利指数基础

### 1.1 凯利的本质

凯利指数反映的是：某家公司对某个结果的赔付态度与市场整体态度的偏差。它是随盘口水位和概率变化而变化的**变量中的变量**，能真实反映各机构给出数据的倾向性和资金流向。

### 1.2 两种计算方法

#### 方法一：简易凯利（main.py 使用此方法）

```
凯利指数 = 某公司该项赔率 / 全部公司该项赔率的算术平均值
```

- 围绕 **1.0** 波动
- > 1.0 = 该公司赔率高于市场均值，赔付慷慨 → 该结果不易打出
- < 1.0 = 该公司赔率低于市场均值，赔付吝啬 → 该结果概率较大
- 优势：只需 API 中多家公司赔率即可计算
- 适用于快速排除法（365 > 1.03 排除、Pinnacle < 0.96 确认等）

#### 方法二：完整凯利（需额外数据，用于深层分析）

```
凯利值 = 赔率 × 隐含概率
```

其中：
- 隐含概率 = (1/赔率) / (1/主胜赔率 + 1/平局赔率 + 1/客胜赔率)
- 围绕**返还率**（约 0.90~0.95）波动，而非 1.0

**更精确的完整凯利需要真实注码比例**（各网站"注码风向标"数据）：

```
实际水位 = 返还率 ÷ 隐含概率
交易值   = 实际水位 × 注码比例
差异     = 交易值 — (赔率 × 注码比例)
```

差异越大（偏负），说明机构在该方向的"真实默认值"与市场表现差距越大，该方向对机构越有利。

> 注意：完整凯利计算中，盘赔本身可能带有诱导性和欺骗性。如果完全根据盘赔来计算凯利值，容易被机构的欺骗性信息诱导。因此完整凯利分析须结合真实注码流向。

### 1.3 核心判断规则（适用于简易凯利）

| 凯利值 | 含义 |
|--------|------|
| < 1.00 | 该公司赔率低于市场平均 → 该结果打出概率较大 |
| = 1.00 | 与市场平均一致 |
| > 1.00 | 该公司赔率高于市场平均 → 该结果不易打出 |

> 凯利指数的本质：衡量某家公司对某个结果的"慷慨程度"。凯利越高 = 赔付越高于市场 = 庄家越不怕赔 = 该结果越不易打出。

## 二、各公司凯利特征与阈值

> **数据源说明**：The Odds API 不提供 SNAI、Interwetten(IT)、澳门、易胜博的数据。
> 以下分为"可直接使用"和"需替代方案"两部分。替代规则权重降低一级，不可作为单独定性依据。

### 2.1 欧赔凯利（重点公司）

**可直接使用：**

| 公司 | 特征 | 报警阈值 | 说明 |
|------|------|----------|------|
| **Bet365** | 行业标杆，自成体系 | **> 1.03** → 该项很难打出 | 365 的凯利超 1.03 是强信号 |
| **Pinnacle** | 最锋利庄家，不限注 | **< 0.96** → 该项易打出 | 赔率最接近真实概率 |
| **威廉希尔** | 英系老牌标杆 | 与 Pinnacle 对比偏差 > 0.05 → 关注 | 离散度分析核心参考 |

**需替代方案（原始庄家不可用）：**

| 原始规则 | 替代方案 | 替代可信度 |
|---------|---------|-----------|
| SNAI 某项凯利 > 1.02 → 该项难打出 | 意甲用 **Codere IT** 凯利辅助判断；其他联赛用 365 > 1.03 规则覆盖 | 中 |
| IT(Interwetten) 某项凯利 1.01~1.02 → 该项易打出 | **Pinnacle 某项凯利 < 0.96** 视为该项易打出 | 中高 |

### 2.1b 全面偏移庄家的凯利修正

**定义**：当某家公司三项凯利**全部 < 1.0**（或全部 > 1.0）时，该公司属于"整体偏移庄家"——其整体返还率与市场平均存在系统性差异，导致凯利绝对值失去参考意义。

**识别方法**：检查该公司主胜、平局、客胜三项凯利是否同侧（全部 < 1.0 或全部 > 1.0）。

**处理规则**：

| 情况 | 处理方式 |
|------|----------|
| 三项凯利全部 < 1.0 | 不可按"< 1.0 = 看好"逐项判断，改为看三项之间的**相对排序和间距** |
| 三项凯利全部 > 1.0 | 不可按"> 1.0 = 排除"逐项判断，同样看相对排序和间距 |
| 最高项与最低项间距 < 0.05 | 该公司对三项结果无明显倾向，凯利信号**严格忽略**（不可作为任何依据，包括辅助参考） |
| 最高项与最低项间距 ≥ 0.05 | 可作为辅助参考（最低项相对更被看好），但权重降低，不可作为定性依据 |

### 2.2 365 + Pinnacle 联合研判（替代原 365+IT 最强信号）

| 365 状态 | Pinnacle 状态 | 判断 |
|----------|--------------|------|
| 两项凯利 > 1.03 | 剩余一项凯利 < 0.96 | 剩余该项打出率极高 |

> 原始规则为"365 两项 > 1.03 + IT 剩余项 1.01~1.02 → 打出率 ≥ 90%"。Pinnacle 替代 IT 后信号强度略降，建议与威廉希尔离散度交叉确认。

### 2.3 亚盘凯利（多公司横向对比）

> 原始规则以"澳门 + 易胜博"为核心。两家均不可用，替代方案：以 **Pinnacle 亚盘**（最锋利）为主，**365 + Marathon** 为辅进行横向对比。

| 场景 | 判断 |
|------|------|
| Pinnacle 盘口比其他公司（365/Marathon）**低半个盘** + 上盘低水 | **上盘极易打出**（尤其主队让球时） |
| Pinnacle 盘口比其他公司**高半个盘** + 下盘低水 | **下盘极易打出** |
| Pinnacle 与 365 盘口方向一致 | 信号增强，可信度高 |
| Pinnacle 与 365 盘口方向相反 | 矛盾信号，**建议放弃或降低置信度** |

> Pinnacle 替代易胜博的角色（亚盘最锋利庄），365 替代澳门的角色（体量大、立场明确）。

## 三、凯利指数实战应用规则

### 3.1 欧赔凯利选项排除法

1. 找出每家公司凯利 > 阈值的项（365 阈值 1.03，Pinnacle 关注 < 0.96 确认项）
2. 被多家公司同时"排除"的选项 → 极难打出
3. 剩余唯一未被排除的选项 → 高概率打出
4. 如果三项凯利都正常（0.97 ~ 1.03） → 凯利无明确指向，需依赖其他分析手段

### 3.2 亚盘凯利热度判断

> 以下统一以**上盘视角**描述，避免上下盘视角混淆。

| 上盘凯利信号 | 盘口配合 | 判断 |
|-------------|----------|------|
| 上盘凯利 < 0.95 + 低水 | 盘口合理 | 上盘打出概率大，庄家压低赔付 |
| 上盘凯利 > 1.05 + 高水 | 盘口合理 | 上盘不被看好，下盘有利 |
| 上盘凯利 > 1.08 | 任意盘口 | 红灯报警，上盘极度危险 |
| 上下盘凯利均接近 1.00 | — | 市场均衡，无明确倾向 |

> **与 2.3 节的关系**：2.3 节描述的是"Pinnacle 盘口与其他公司的差异"，本节描述的是"单家公司凯利值本身的高低"。两者可交叉验证：如果 Pinnacle 盘口低于其他公司（2.3 规则）且上盘凯利 < 0.95（本节规则），双重确认上盘打出概率高。

#### 3.2b 受让盘（+盘）凯利的解读边界

> **关键警告**：受让盘的凯利信号只能用于亚盘方向判断，**不可直接推导胜平负**。

**原因**：受让盘"赢盘"包含多种胜平负结果。例如受让 +0.75 赢盘 = 该队赢 / 平 / 小负1球（赢半）。凯利低只意味着"不会大败"，不等于"该队胜场"。

**正确用法**：

| 盘口类型 | 凯利低的含义 | 可推导 | 不可推导 |
|----------|------------|--------|---------|
| 让球盘（-盘） | 让球方赢盘 | 让球方赢球 ≈ 胜场 | — |
| 受让盘（+盘） | 受让方赢盘（不大败） | 亚盘方向 | ~~受让方胜场~~（错误） |

**实战规则**：当受让方凯利极低（如 < 0.93）时：
1. 亚盘方向：受让方赢盘（高置信度）
2. 胜平负：必须独立推演——结合欧赔凯利、返还率、基本面单独判断，不可从亚盘凯利直接继承结论
3. 受让方赢盘但让球方赢球（如 0-1 小负赢半盘）是完全合理的结果，不可忽视

#### 3.2c 临场升盘后受让方凯利转低的二次误判

**触发条件**：临场④盘口从 -0.75 升至 -1.0（或类似级别），此时受让方亚盘水位变低、凯利 < 1.0，切勿直接推导为"下盘胜"。

**实战案例**：2026-04-19 弗赖堡 2:1 海登海姆（德甲）
- 临场④ Pinnacle 升盘至 -1.0，客队水位 1.83（低水），客方凯利 0.977
- 初看是"下盘赢盘信号"，分析将其等同于"下盘胜/平首选"
- 实际弗赖堡 2:1 赢球，受让 +1.0 **走盘**（海登海姆输 1 球正好等于让球深度）
- 教训：升盘后受让方凯利转低只意味着"上盘难以覆盖新的让球深度（-1.0）"，而非"上盘不赢球"。当上盘主场+实力占优时，**上盘赢 1 球（走盘）是最常见结果**

**规则补充**：临场升盘情境下，受让方凯利 < 1.0 的正确解读：
- 亚盘方向：受让方不输盘（可能赢盘或走盘）
- 胜平负：**不可排除上盘小胜**，尤其当上盘基本面强势时，比分预测应将"上盘赢 1 球"列为首选

### 3.3 临场凯利变化（最重要）

> **临场凯利比初盘凯利更有参考价值**。庄家经常在临场阶段变盘，初盘凯利可能失效。

| 临场变化 | 含义 |
|----------|------|
| 某项凯利从 < 1.00 升至 > 1.03 | 庄家临场放弃保护该方向 → 该项不出 |
| 某项凯利从 > 1.03 降至 < 1.00 | 庄家临场收紧赔付 → 该项可能打出 |
| 临场凯利剧烈波动 | 知情资金入场，需警惕 |

### 3.4 Pinnacle 亚盘主凯利持续 < 0.90 的信号持久性

**规则**：当 Pinnacle 亚盘主凯利在初盘阶段持续低于 0.90（连续 2 个及以上节点），该信号**不因后续升盘动作而消解**，应维持"上盘真实看好"的判断。

**错误做法**：庄家在中盘升盘后，主凯利回升至 1.0 附近，误判为"信号消解，转为诱盘"。

**正确解读**：

| 场景 | 含义 |
|------|------|
| 初盘主凯利持续 < 0.90 → 中盘升盘 → 主凯利回升至 1.0 附近 | 升盘是庄家**修正定价**（初盘低估了上盘强度），主凯利回升是盘口调整的自然结果，不是信号反转 |
| 升盘后主凯利稳定在 0.96~1.02 | 上盘真实看好信号仍有效，只是强度从"极强"降为"中强" |
| 升盘后主凯利升至 > 1.05 | 信号方向反转，需重新评估 |

**判断流程**：
1. 初盘阶段 Pinnacle 亚盘主凯利是否持续 < 0.90？（连续 2 节点）
2. 是 → 锁定"上盘真实看好"基准信号
3. 中盘升盘后主凯利是否升至 > 1.05？
   - 否（回升至 0.96~1.05）→ 基准信号维持，升盘 = 修正定价
   - 是（升至 > 1.05）→ 信号反转，重新评估

## 四、凯利与其他分析手段的配合

### 4.1 凯利 + 欧亚对比

| 欧赔凯利信号 | 亚盘信号 | 综合判断 |
|-------------|----------|----------|
| 365主胜凯利 < 0.97 | 上盘低水 + 让球合理 | 强信号：上盘打出 |
| 365主胜凯利 > 1.03 | 上盘升水 + 盘口开大 | 强信号：上盘不出 |
| 365主胜凯利 < 0.97 | 上盘高水 + 盘口开大 | **欧凯亚盘背离** → 庄家诱盘可能，谨慎 |

#### 4.1b 凯利与返还率冲突时的处理规则

当亚盘凯利方向与返还率变化方向不一致时，不可让单一凯利信号压过返还率信号。

| 冲突场景 | 处理方式 |
|----------|----------|
| 亚盘凯利指向 A 方向 + 返还率上升且低水方为 B 方向 | **降低置信度至 65 以下**，A 和 B 方向给予同等权重 |
| 亚盘凯利指向 A 方向 + 返还率稳定或下降 | 无冲突，凯利信号有效 |
| 亚盘凯利指向 A 方向 + 返还率上升且低水方也为 A 方向 | 双重确认，凯利信号增强 |

**返还率上升的含义**：庄家在压缩利润吸筹，通常低水方（赔率最低的一方）获益。当返还率上升 ≥ 1.5% 时，这是一个中等强度的方向性信号，不可忽视。

### 4.2 凯利 + 威廉/Pinnacle 对比法

> 原始规则为"威廉/IT 对比法"，IT 不可用，用 Pinnacle 替代。

| 规则 | 判断 |
|------|------|
| 威廉某项赔率明显高于 Pinnacle → 且该项凯利 > 1.03 | 该结果大概率打不出（双重确认） |
| Pinnacle 某项凯利 < 0.96 → 且 365 其余两项凯利 > 1.03 | 该项高概率打出（三重确认） |

## 五、深盘博高水的凯利辅助

遇到深盘（一球以上）想博高水单选时，可结合欧赔凯利：

1. 找出各公司平局和客胜的凯利值
2. 如果某公司平局凯利 < 0.98 且客胜凯利 > 1.05 → 平局有价值
3. 如果某公司客胜凯利 < 0.95 且主胜凯利 > 1.05 → 冷门有价值
4. 必须结合基本面（伤停、状态、战意）交叉验证

## 六、完整凯利深层分析（进阶）

> 以下方法需要真实注码比例数据（来自足彩网站的"客户调查"/"注码风向标"），API 无法提供。
> 适用场景：当简易凯利无法给出明确判断时，可结合注码比例做深层推算。

### 6.1 分析流程

以曼联 0:1 利兹联（足总杯）为例：

**第一步：计算隐含概率（从赔率推导）**

| 公司 | 主胜赔率 | 平局赔率 | 客胜赔率 | 返还率 |
|------|----------|----------|----------|--------|
| 立博 | 1.25 | 6.00 | 10.00 | 93.7% |
| 威廉 | 1.22 | 6.00 | 12.00 | 93.8% |

立博隐含概率：主胜 75.0%、平局 15.6%、客胜 9.4%

**第二步：获取注码比例（从风向标调查）**

主胜 80%、平局 12%、客胜 8%（综合 2~3 家网站取平均）

**第三步：用注码比例计算交易值**

```
市场交易值 = 赔率 × 注码比例
```
- 主胜：1.25 × 80% = **1.00**
- 平局：6.00 × 12% = **0.72**
- 客胜：12.00 × 8% = **0.96**

**第四步：用返还率重算"真实默认值"**

```
实际水位 = 返还率 ÷ 隐含概率
真实默认值 = 实际水位 × 注码比例
```

> 此处使用行业基准返还率（约 90%）而非该公司实际返还率（93.7%），目的是剥离庄家利润后还原"真实概率对应的赔付水平"。
> 如果用实际返还率（93.7%），计算出的是"含利润的默认赔付"，无法反映机构的真实倾向。

- 主胜：(90% ÷ 75%) × 80% = 1.20 × 80% = **0.96**
- 平局：(90% ÷ 15.6%) × 12% = 5.76 × 12% = **0.69**
- 客胜：(90% ÷ 9.4%) × 8% = 9.59 × 8% = **0.77**

**第五步：对比差异**

| 结果 | 真实默认值 | 市场交易值 | 差异 |
|------|-----------|-----------|------|
| 主胜 | 0.96 | 1.00 | **-0.04** |
| 平局 | 0.69 | 0.72 | **-0.03** |
| 客胜 | 0.77 | 0.96 | **-0.19** |

**结论**：主胜交易值 1.00，综合（市场交易值+真实默认值）/2 = 0.98，偏高。客胜差异 -0.19 最大 = 机构在客胜方向最获利。注码严重倾斜至主胜（80%），最终曼联大热输球。

### 6.2 差异判读规则

| 差异幅度 | 含义 |
|----------|------|
| 差异 > -0.05 | 机构利润正常，该方向中性 |
| 差异 -0.05 ~ -0.15 | 机构利润偏大，该方向对机构有利 |
| 差异 < -0.15 | 机构利润丰厚，该方向极度有利 → 对面方向不易打出 |

### 6.3 综合凯利值计算

```
综合凯利 = (市场交易值 + 真实默认值) / 2
```

| 综合凯利值 | 判断 |
|-----------|------|
| > 0.98 | 偏高 → 该方向过热，不利 |
| 0.92 ~ 0.98 | 中性区间 |
| < 0.92 | 偏低 → 该方向对机构有利 → 不易打出 |

> **关键提醒**：注码比例应取 2~3 家网站的平均值，单一来源可能存在偏差。

## 七、使用注意事项

1. **临场数据最重要**：看不到临场凯利时谨慎使用
2. **联赛适用性**：周末德甲（22:30）和英超（23:00）可重点关注
3. **凯利不是万能**：必须结合基本面、伤停状态、球队战意等因素
4. **不同公司阈值不同**：365 用 1.03，Pinnacle 用 0.96（反向），不可混用
5. **亚盘凯利**：以 Pinnacle 为核心参考（替代原澳门/易胜博角色），365 和 Marathon 为辅
6. **简易凯利的局限**：盘赔本身可能带有诱导性，简易凯利无法反映真实注码流向，遇到分歧时应结合完整凯利分析（第六章）
7. **替代规则降权**：所有使用替代庄家的规则，权重降低一级，不可作为单独定性依据
8. **Betfair 交易所凯利须特殊处理**：Betfair 是交易所（撮合投注者对赌），不是传统庄家（庄家 vs 投注者）。其赔率由散户投注供需决定，不存在"庄家主动调整赔付"的操盘行为。因此：
   - Betfair 某项凯利大幅下降 = 散户资金涌入该方向 → 应视为该方向**大热信号**，而非"庄家收紧赔付看好该方向"
   - 传统庄家凯利下降 = 庄家主动降低赔付 → "看好"信号；Betfair 凯利下降 = 市场追捧 → "大热"信号。两者解读方向**相反**
   - Betfair 凯利不适用第三章 3.3 节"临场凯利变化"规则（该规则假设存在庄家主动操盘）
   - Betfair 凯利可用于判断市场热度方向，但不可用于判断庄家意图
9. **单一强信号不可主导结论**：当算法 B（操盘手法）与算法 C（风控指数）存在冲突时，即使某个凯利信号极强（如亚盘凯利 < 0.93），也不可让它压过其他冲突信号（如返还率方向相反、反诱场景匹配）。B/C 冲突时必须降低置信度至 65 以下，并给冲突双方同等权重。单一信号越强越容易产生"锚定效应"，反而需要更警惕其他方向的证据
10. **平局凯利 < 0.96 = B/C 冲突信号**：当算法 B 推演为某方胜出，但有公司平局凯利 < 0.96（报警阈值），此为 B/C 冲突，**不可定性为"基本一致"**。平局凯利报警意味着该公司认为平局概率被市场低估，必须降低置信度 10~15 分，并将平局至少作为次选纳入最终结论。此规则与 H2H 高平局率（≥ 60%）叠加时，平局应直接升为首选（参考 reference_case_lessons.md 第八章）。案例：霍芬海姆 3-3 斯图加特（2026-05-02 德甲），Tipico 平局凯利 0.935，H2H 近 5 场 80% 平局率，两信号叠加本应将平局列为首选，却被定性为"B/C 基本一致"推主胜，严重失误

## 八、返还率判别规则（算法 C-2）

> 返还率 = 1 / (1/主胜赔率 + 1/平局赔率 + 1/客胜赔率)，正常范围 90% ~ 96%。

| 规则 | 判断 |
|------|------|
| 临场返还率整体上升（如 92% → 94%） | 庄家看好某方并降水吸筹，通常是低水方获益 |
| 临场返还率整体下降 | 庄家提高利润，对赛果无明确倾向或规避风险 |
| 主胜单项返还率远低于平/负（主胜隐含概率 × 赔率显著偏低） | 庄家在压缩主胜利润 → **主胜可信** |
| 单项返还率异常偏高 | 该项庄家赔付压力大 → 不利于该结果打出 |
| 返还率上升 ≥ 1.5% + 低水方为某方向 | 与亚盘凯利冲突时，置信度降至 65 以下，两方向给予同等权重（见 4.1b 节） |
