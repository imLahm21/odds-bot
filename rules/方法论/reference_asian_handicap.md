# 亚盘操盘手法与实战盘路参考手册

> 本文件供 LLM 分析赔率数据时作为规则库参考。

---

## 一、亚盘基础概念

### 1.1 盘口（让球）

盘口是平衡两队实力差距的砝码。强队让弱队球，差距越大让球越多。

**常见让球级别**（从小到大）：
- 平手 → 平半（平手/半球）→ 半球 → 半一（半球/一球）→ 一球 → 一球/球半 → 球半 → 球半/两球 → 两球 → 两球以上

**大盘与小盘划分**：
- 小盘：半球/一球（含）以下
- 大盘：一球以上（不含一球，即一球/球半起）

> 一球盘属于小盘与大盘的临界点，归入小盘。实战中一球盘兼具两类特征，需结合水位和变盘方向综合判断。

**主客场换算规则**：
强队做客时让球通常比主场降两级：
- 主场让一球 → 客场让半球（一球 / 半一 / 半球）
- 主场让半一 → 客场让平半（半一 / 半球 / 平半）

### 1.2 水位（贴水）

水位是结算系数，通常在 0.80 ～ 1.05 之间。

**关键水位阈值**：
| 水位区间 | 术语 | 含义 |
|----------|------|------|
| < 0.80 | 超低水 | 该方向资金极度集中，庄家极不愿赔付 |
| 0.80 ~ 0.87 | 低水 | 庄家偏向该方向胜出 |
| 0.87 ~ 0.95 | 中水 | 正常受注范围 |
| 0.95 ~ 1.05 | 高水 | 该方向不被庄家看好 |
| > 1.05 | 超高水 | 庄家认为该方向极难打出 |

> 双方水位之和通常在 1.80 ～ 1.90 之间，差额即庄家直观利润。

### 1.3 实力盘与蛊惑盘

**实力盘**：按球队联赛排名合理开盘。
- 上游（1~4）vs 中上游（5~9）：让半球到半一
- 同等级主场：让平半到半球
- 跨一级：让半球到半一，以此类推

**蛊惑盘**：盘口与实力排名不符，庄家先入为主控制热方资金。

> **核心原则**：分析时不要先看盘口，应先看基本面（近况、主客场战绩、名气实力、交锋往绩、伤停、积分形势、赛程战意），再对照盘口是否合理。

## 二、盘口与水位变动四大规则

### 规则一：升盘降水看时间

升盘又降水 = 庄家已获取非公开信息或初盘判断失误。

| 时间 | 判断 |
|------|------|
| **早升早降**（受注初期） | 初盘失误修正，上盘打出概率大 |
| **临场升盘持续降水**（开赛前1~2小时） | 上盘过热失控，大热必死，下盘为主（可能赢球输盘） |

### 规则二：降盘降水观走势

降盘降水 = 换汤不换药，需看水位走势图。

| 走势形态 | 判断 |
|----------|------|
| 抛物线凹形 或 斜线向上 | 上盘打出概率大 |
| 抛物线凸形 或 斜线向下 | 下盘打出概率大 |

### 规则二补充：升盘后客水大幅下降 = 真实看好上盘

升盘后若**客水（下盘水位）大幅下降（≥ 0.15）**，是庄家真实看好上盘的强信号，不可误判为诱盘。

| 升盘后客水变化 | 含义 | 判断 |
|-------------|------|------|
| 客水下降 ≥ 0.15（如 2.07 → 1.83） | 庄家收紧下盘赔付，减少下盘赔付风险 | **真实看好上盘，上盘打出** |
| 客水下降 < 0.08 或维持不变 | 庄家对下盘赔付无明显控制 | 升盘性质待定，需结合其他信号；**但若同时欧赔主胜持续下降，则欧亚联合信号优先，按 reference_case_lessons.md 第一章规则 1 确认为真实看好上盘** |
| 客水下降同时主水也下降 | 双边收紧，庄家在平衡资金 | 参考主水降幅，降幅更大方向为庄家真实看好方 |

**与诱盘的关键区分**：
- 诱上型升盘：升盘后主水维持高水（≥ 2.05），客水无明显变化 → 庄家未收紧下盘赔付 → 诱盘
- 真实看好上盘：升盘后客水大幅下降 → 庄家主动压缩下盘赔付空间 → 真实看好上盘

### 规则三：升盘升水走水位

升盘升水 = 庄家调节资金平衡。

| 水位上升幅度 | 判断 |
|-------------|------|
| 升盘水位上升 < 10个百分点 | 上盘打出概率大 |
| 升盘水位上升 > 10个百分点（尤其超高水） | 下盘打出概率大（**含"上盘赢球输盘/走盘"**，非必然下盘胜） |

### 规则四：降盘升水多开上

降盘升水 = 正诱盘（庄家冒险操盘），比较少见。
- 盘口下降 + 上盘水位上升 → 让人感觉有假而转投下盘
- **结果多为上盘赢盘**

## 三、大小盘升降判断

| 场景 | 判断 |
|------|------|
| **大盘升盘**（临场） | 下盘 ≥ 60% 概率（含赢球输盘），热升居多，注意大冷门 |
| **大盘降盘** | 勇敢去上盘 |
| **小盘降盘**（临场） | 上盘为佳，反诱盘（弱队为让球方时尤其明显，准确率 ≥ 85%） |
| **小盘升盘** | 下盘概率最高，庄家最常见操盘手法——为上盘造势后诱杀 |

### 3.1 初盘开大 + 连续降盘的军规匹配澄清

当初盘开大（实际盘口 > 理论盘口）后出现**连续降盘**（如 -0.75→-0.50→-0.25），必须走军规 #3（修正）路径，**不可误套军规 #4（诱上）**。

**核心逻辑**：降盘 = 降低上盘赢盘门槛 = 对上盘有利。军规 #4 的前提是"不变盘"，一旦发生降盘即转入修正逻辑。

| 初盘开大后的变化 | 军规匹配 | 后续判断 |
|-----------------|---------|---------|
| 不变盘 + 上盘大热 + 降水至低水 | #4（诱上→下盘胜） | — |
| 早中期降盘至合适 | **#3（修正→转场景三）** | 场景三规则接管 |
| 临场连续降盘（≥2次） | **#3→场景三→#10（上盘赢球赢盘）** | 降盘 = 庄家修正初盘高估，看好上盘小胜 |
| 临场降盘 + 欧赔主胜同步上升 | **欧亚同步降盘 = 定价修正** | 上盘赢球但净胜球少（参考 reference_dynamic_analysis.md 2.7 节） |

## 四、特殊盘型识别

### 4.1 死水盘

**定义**：从初盘到收盘，水位始终维持超高水（>1.05）和超低水（<0.80），全程无波动。

| 条件 | 结论 |
|------|------|
| 全程死水不变 | 超低水方胜出概率 ≥ 90% |
| 全程死水，但**临场超高水方水位出现波动或小幅下降** | 超高水方胜出概率 ≥ 80% |

### 4.2 蛙跳盘

**定义**：受注过程中连续三次同方向变盘。

| 变化 | 结论 |
|------|------|
| 半球 → 半一 → 一球（连升三级） | 上盘为主 |
| 半球 → 平半 → 平手（连降三级） | 下盘为主 |

> 受注期间准确率高，临场连续变盘则准确率稍低。

### 4.3 V字型走势

上盘水位呈"V"字型走势 → **上盘不利**。

> 原始规则特指澳门水位，The Odds API 不提供澳门数据。替代方案：观察 Pinnacle 或 365 亚盘水位是否呈 V 字型走势，判断逻辑相同。

### 4.4 波动盘

水位一上一下波动很大，临场30分钟后下盘水位仍在上升 → **下盘有利**（考虑标平）。

## 五、四大诱盘手法

> 原始规则以澳彩（澳门）为主要研究对象。

### 5.1 正诱（诱闲家买看似正确的一方）

**操盘特征**：
- 开小盘口 → 受注后上盘升盘中水或超低水 → 上盘受热 → 造看好上盘之势
- 或顺欧赔开盘给高水 → 不断降盘降水 → 明目张胆诱上

**结果**：上盘打不出，球队不胜

**适用场景**（初级彩民多的赛事）：
- 周六周日、晚间前半夜
- 五大联赛
- 赛前消息利好强队 / 强队换帅 / 历史交锋明显占优
- 比赛日首场或同时开打的场次
- 球队有明确目标而对手无欲无求
- 两队近期战绩对比鲜明

### 5.2 反诱（让中高级彩民自作聪明）

**操盘特征**：
- 强队开浅盘 → 大热低水跑出（看似过热，实则庄家有意）
- 或强队开深盘 → 升盘不降水 → 让高级彩民认为过热而转投弱队

**结果**：强队仍然胜出

**适用场景**（中高级彩民多的赛事）：
- 凌晨1~2点后的较晚比赛
- 同一联赛中有 2~3 场类似或反向做盘（交叉盘）
- 二级联赛、英冠联赛
- **欧冠/杯赛淘汰赛**（尤其次回合）：中高级彩民集中度极高，反诱概率显著上升

#### 5.2b 淘汰赛次回合反诱特殊规则

**触发条件**：杯赛/欧冠淘汰赛次回合，一方首回合领先 ≥ 2 球。

**操盘特征**：
- 领先方主场，让球方（对手）开浅盘（低于理论盘口）
- 让球方大热（"必须追分"题材驱动资金涌入）
- 盘口全程不变或仅微调，庄家不主动造势

**陷阱逻辑**：
- 中高级彩民看到"让球方大热 + 浅盘"→ 判断为浅诱 → 转投领先方
- 但实际上让球方确实有进攻能力和战术动力（背水一战）→ 让球方仍然赢球
- 领先方虽然晋级目标明确，但"保守防守"并不等于"不丢球"

**与浅诱的关键区分**：浅诱中上盘大热不出（赢球输盘或不胜）；反诱中上盘大热仍然打出（强队仍然胜出）。区分依据见 5.4b 鉴别决策树。

> 案例：马竞 1:2 巴萨（2026 欧冠1/4次回合），马竞首回合客场 2-0 领先。巴萨让平/半（+0.75），理论盘口应为半球（+0.5），初盘开小。巴萨大热，盘口全程不变。中高级彩民判断为浅诱转投马竞，但巴萨仍然 2-1 赢球。马竞受让 +0.75 赢半盘（亚盘方向正确），但胜平负方向为巴萨胜。

### 5.3 深诱（针对中高级彩民的深度陷阱）

**操盘特征**：
- 强队或中上游强队让盘开大
- 弱队主场对强队仅开平手低水或平半（理应让更大）
- 欧赔对应给低水，澳彩给中高水后不断降至中低水

**适用场景**：
- 比赛较少时段、凌晨比赛
- 二级联赛、杯赛
- 排名相近但近况/客场战绩差异明显的对阵

### 5.4 浅诱（五大联赛最常用）

**操盘特征**：
- 故意开小盘口（低于理论盘口半个到一个盘）
- 上盘看似便宜、容易打出 → 资金涌入上盘
- 水位维持中低水区间，不升盘不调整，静等资金自然聚集

**结果**：上盘大热不出（赢球输盘或直接不胜）

**与正诱的区别**：
- 正诱 = 主动造势（升盘、降水等明显动作引导资金）
- 浅诱 = 被动等待（仅靠盘口本身的便宜感吸引资金，操盘痕迹最少）

**适用场景**：
- 五大联赛周末场次
- 强队有明显利好题材（近期连胜/历史交锋优势）
- 初盘开出后市场自然形成一边倒预期

#### 5.4b 浅诱与反诱鉴别决策树

当出现"初盘开小 + 上盘大热 + 不变盘"时，浅诱和反诱的操盘表象几乎相同，必须通过以下决策树区分：

**第一步：判断赛事环境**

| 条件 | 倾向 |
|------|------|
| 五大联赛周末场次 + 晚间前半夜 | **浅诱**（初级彩民为主） |
| 凌晨 01:00 后 + 欧冠/杯赛淘汰赛 | **反诱**（中高级彩民为主） |
| 二级联赛凌晨场 | **反诱** |

**第二步：判断上盘基本面强度**

| 条件 | 倾向 |
|------|------|
| 上盘近 5 场 ≥ 4 胜 + 客场能力强 + 有明确战术动力（如必须追分） | **反诱**（上盘有真实打出能力） |
| 上盘状态一般或有利空 + 仅靠名气/历史交锋受热 | **浅诱**（上盘受热缺乏基本面支撑） |

**第三步：检查返还率方向**

| 条件 | 倾向 |
|------|------|
| 返还率上升 ≥ 1.5% + 低水方为上盘（让球方） | **反诱**（庄家降水吸筹方向与上盘一致） |
| 返还率稳定或下降 | **浅诱**（庄家无明确吸筹动作） |

**决策规则**：三步中有 2 步及以上指向反诱 → 按反诱处理（上盘仍可能打出）；否则按浅诱处理。

> **核心教训**：不要因为"开小+大热+不变盘"就机械套用军规 #7（浅诱→下盘胜）。赛事环境和上盘基本面强度是区分浅诱与反诱的关键变量。

## 六、盘口受热方向判断

| 盘口 | 水位变化 | 受热方 | 庄家意图 |
|------|----------|--------|----------|
| 开大盘，不变盘 | — | 下盘大热 | 看好上盘，诱下盘 |
| 开大盘，临场降盘 | — | 下盘大热 | 看好上盘，诱下盘 |
| 开大盘，早中期降盘 | — | — | 修正为合适盘口 |
| 开大盘，不变盘 | 降至低水 | 上盘大热 | 诱上盘 |
| 开大盘，临场升盘 | — | 上盘大热 | 诱上盘 |
| 开大盘，早中期升盘 | — | 上盘大热 | 诱上盘 |
| 开小盘，不变盘 | — | 上盘大热 | — |
| 开小盘，早中期升盘 | — | — | 修正为合适盘口 |
| 盘口合适，不变盘 | — | 哪方受热 | 哪方有难（热死原则） |
| 盘口合适，临场降盘（主水↓或持平） | — | — | 减小上盘赔付，赢球赢盘（军规 #10） |
| 盘口合适，临场降盘（主水反升） | — | — | **不符合 #10**，需重新评估（参考 reference_case_lessons.md 第七章） |
| 盘口合适，升盘 | — | 上盘受热 | 热上盘 |
| 盘口合适，早期降盘 | — | — | 转为盘口开大 |
| 盘口合适，早期升盘 | — | — | 转为盘口开小 |

## 七、常用盘口速判规则

| 盘口特征 | 速判 |
|----------|------|
| 主队让平半高水 | 主队信心不足，诱上盘 → 下客队 |
| 主队让半球低水 | 庄家对主队有信心 → 下主队 |
| 主队让半球高水 | 庄家对主队信心不足 → 下客队（一般不能要主队） |
| 主队让半一低水 | → 下主队 |
| 初盘主队让半一 → 改让一球 | → 下客队 |
| 客队初盘平手低水 → 改让平半高水 | → 下客队 |
| 开半一的盘 | 一般不能要下盘 |
| 开一球或一球球半的盘 | 一般不能要上盘 |
| 开一球半以上的盘 | 一般不能要下盘 |

## 九、欧盘辅助判断（威廉希尔对比法）

> 原始规则基于"威廉希尔 vs Interwetten(IT)"对比法。The Odds API 不提供 Interwetten 数据。
> **替代方案**：用 **Pinnacle**（最锋利庄家）替代 IT 的角色，核心逻辑不变——比较"传统英系庄"与"锋利庄"的赔率差异。

| 规则 | 判断 |
|------|------|
| 威廉某项赔率明显高于 Pinnacle 对应项 | 该结果大概率打不出 |
| 威廉某项赔率低于 Pinnacle 且利润差 < 0.2 | 该结果最可能打出 |
| Pinnacle 变盘后最低项与威廉一致 | 高概率出该结果 |
| 威廉变盘后与 Pinnacle 一致 | 该结果不易打出 |
| Pinnacle 变盘且变低项与威廉一致 | 出低盘方向 |

> 欧盘判断须结合亚盘交叉验证。Pinnacle 替代 IT 后信号强度略降，建议增加 365 作为第三方验证。

## 十、核心纪律

1. **深盘不碰**
2. 低级别联赛只玩平手盘
3. 一球盘跟大球
4. 五大联赛专注：平手盘、平手升平半、平半降平手、半球退平半、半一升一球/降半球、一球盘
5. **放弃下盘 = 放弃 50% 的机会**——不要有上盘和强队情结
6. 球无假球，盘有假盘——保持中立观点
7. 先看基本面，再对照盘口
8. **亚盘赢盘 ≠ 胜场**——受让盘（+盘）凯利低只意味着"不会大败/赢盘"，不等于"该队胜场"。胜平负判断必须独立推演，不可直接从亚盘方向推导
9. **基本面不可反向扭曲盘口解读**——先看基本面再对照盘口（第 7 条），但不可让基本面预设（如"领先方必然保守"）反过来影响盘口信号的客观解读。盘口信号与基本面预设矛盾时，以盘口信号为准

## 十四、异常信号速查

| 异常现象 | 可能含义 |
|----------|----------|
| 某家庄赔率与其他庄差异 > 0.2 | 该庄有异常信息或数据不同步 |
| 临场前 3 小时水位突然剧变 | 知情资金入场 |
| 让球变化超过 0.5 个球 | 极不寻常，关注伤病/停赛等场外因素 |
| 平局赔率跌破 3.0（弱旅之间） | 可能存在赛前谈判信号 |
| 欧赔不动但 Pinnacle 独自降盘 | 庄家对上盘没信心（参考本文件 #15 #17） |
| 临场返还率突然飙升 > 2% | 庄家在特定方向大幅降水吸筹 |
| 主流公司凯利某项高度一致且 < 0.95 | 该结果打出概率极高 |
| 算法 B 推演为上盘但凯利报警上盘 | **B/C 冲突** → 降低置信度，优先考虑凯利风险 |
