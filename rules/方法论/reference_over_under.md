# 大小球（总进球数）操盘手法与实战盘路参考手册

> 本文件供 LLM 分析赔率数据时作为大小球盘口的规则库参考，体例与 [reference_asian_handicap.md](reference_asian_handicap.md) 对齐。
> 资料来源：百盈足球大小球系列、虎扑「西凉夏天」专栏、走地 20 分钟法则、Pinnacle/Covers 英文换算资料综合整理。

## 一、大小球基础概念

### 1.1 盘口与赔付规则

大小球以全场常规 90 分钟（含伤停补时，不含加时/点球）总进球数判定。
盘口分整数盘、半球盘、四分盘三类：

| 盘口类型 | 示例 | 走盘可能 | 说明 |
|----------|------|----------|------|
| 整数盘 | 2、3 | 有走盘 | 总进球 = 盘口 → 退还本金 |
| 半球盘 | 2.5、3.5 | 无走盘 | 一翻两瞪眼，全赢或全输 |
| 四分盘 | 2.25、2.75 | 有半赢/半输 | 拆成相邻两盘各下一半 |

**四分盘拆分逻辑**（与亚盘四分盘同理）：
- 大 2.25 = 一半下「大 2.0」+ 一半下「大 2.5」
- 大 2.75 = 一半下「大 2.5」+ 一半下「大 3.0」
- 小盘方向相反。

### 1.2 水位术语（与亚盘一致）

水位细分标准（三大区间 + 七档细分）：

| 大区间 | 细分档 | 水位区间 | 含义 |
|--------|--------|----------|------|
| **（1）低水区间**（< 0.85） | 超低水 | < 0.75 | 该方向（大或小）被庄家极度看好 |
| | 低水 | 0.75 ~ 0.85 | 该方向被庄家看好 |
| **（2）中水区间**（0.86 ~ 1.00） | 中低水 | 0.86 ~ 0.90 | 该方向略受看好 |
| | 中水 | 0.91 ~ 0.95 | 正常受注 |
| | 中高水 | 0.96 ~ 1.00 | 正常偏冷（1.00 有时也可视为高水） |
| **（3）高水区间**（≥ 1.00） | 高水（飘红） | 1.00 ~ 1.08 | 该方向不被看好 / 庄家诱导该方向 |
| | 超高水 | > 1.08 | 该方向极度不被看好 / 强诱导信号 |

### 1.3 大小球与亚盘的本质差异（核心理念）

1. **大小球基本盘 = 亚盘的标准盘**，对应的「标准盘」就是全场进球总数。
2. **全场进球四区段**：0~1 球、2~3 球、4~6 球、7+ 球。现代足球赛果最多落在 **2~3 球**，这是开盘的杠杆基准。
   - 盘口 ≥ 2.75 时，2~3 球偏「小球」；
   - 盘口 ≤ 2.25 时，2~3 球偏「大球」。
3. **大小球的升降水「假的多」**：亚盘升降水多为真实资金，大小球升降水多为诱导。
   → **大球盘更看重开盘位置，而非临场走向**；很多盘根本不变，不变比例远高于亚盘。
4. **决定大小的是「丢球能力」而非进攻能力**：矛与盾，烂盾比利矛更容易出大球。豪门互拼常浪费时间，保级烂队互踢反而狂进球。

## 二、初盘定性与开盘思路（赛前判断核心）

> **核心原则**：初盘参考价值远高于临场。初盘是机构综合所有基本面后的结论，临盘只是迎合资金/比赛做的调整。

### 2.1 初盘开大 / 开小 / 合适

先用基本面估算「合理盘口」（见第三章），再与实际初盘对比：

- 实际初盘 > 合理盘口 → **开大**（深开，机构倾向防大球 / 看好小球）
- 实际初盘 < 合理盘口 → **开小**（浅开，机构压低小球赔付风险）
- 实际初盘 ≈ 合理盘口 → **合适**

### 2.2 机构调盘的双重动机（区别于亚盘）

亚盘调盘 = 仅为平衡两边注码、稳赚抽水。
**大小球调盘 = 平衡注码 + 与玩家对赌**。

> 玩家有「追大不追小」的潜意识（大比分好看刺激），买大比例天然偏高，机构掐准这一心理在盘口上做文章。**降盘常诱小、升盘常诱大，方向多与表象相反。**

### 2.3 2.25 球初盘实战规律（虎扑西凉夏天专栏 + 北单统计）

> 统计基础：32 场四家初盘均为 2.25 球的比赛，打出 2~3 球的有 21 场，占 65.6%。

| 序号 | 盘型条件 | 高概率赛果 |
|------|----------|-----------|
| 1 | 初盘 2.25 球**低水**（≤0.85），后期不变盘；或利记赛前降到 2 球低水 | **小于 2 球** |
| 2 | 初盘 2.25 球**中水**（0.85~1.00），后期不变盘，水位飘红 | **3 球以上** |
| 3 | 初盘 2.25 球，澳门岿然不动，易胜博/韦德/利记降 2 球 | **正好 2 球** |
| 4 | 初盘 2.25 球，澳门(或不动)、易胜博/韦德/利记升 2.5 球 | **正好 2 球** |
| 5 | 初盘 2.25 球，后市升 2.5 球，但威廉/立博/马会下调 2.5 球水位 | 诱大明显，2.5 大概率小，**2 球最大** |

> 解读：澳门（单盘、无副盘、返水低、敏感）岿然不动 = 真实定价锚点；其余机构去诱小/诱大，正是反向信号。打出 2 球走水时机构 0 抽水反而收益最大，故「降盘诱小」常落 2 球。

### 2.4 不要碰 2.5 球初盘且后市不动的场

机构统一开 2.5 球初盘且后市无变动 = 机构对大小持模糊态度（连机构都判不准）→ 普通玩家应回避。
但若 2.5 球初盘后市**有变动**，则可结合变动端倪研判（升 2.75 防大不足 → 反易破大；降 2.25 → 看小）。

## 三、合理盘口估算（基本面 → 大小球）

### 3.1 攻守数据法（主队主场 + 客队客场）

看主队主场得失球数 + 客队客场得失球数：
- 攻强守弱（得失球都多，如场均合计偏高）→ 易大球
- 攻守平衡 / 攻弱守强 → 易小球
- **关键看「下盘（弱队）进球能力」**：弱队 0 球水位越高 → 越可能有进球 → 利大球

### 3.2 进球数预估公式（网易/传统法，仅作辅助）

```
近期平均得失球 = (主队近 N 场总进失球 + 客队近 N 场总进失球) ÷ 总场次
预估进球 = (近期平均得失球 + 双方交战平均进球) ÷ 2
```

> 重大比赛踢法转保守，需向下修正。公式仅用于反查机构开盘是否有明显偏差。

### 3.3 影响大小球的客观因素（开盘无法左右）

| 因素 | 倾向 |
|------|------|
| **联赛球风** | 大球：荷甲/荷乙、德甲、瑞超、挪超、日职、巴甲；小球：法甲/法乙、意甲/意乙、西乙、俄超 |
| **丢球能力** | 烂盾 > 利矛，决定大小的首要因素 |
| **天气** | 寒冷/大风/雨雪 → 效率下降 → 偏小 |
| **赛制** | 赛会制（世界杯等）偏小球；联赛密集双赛/赛季末无欲求 → 偏大球 |
| **战意** | 双方都要 3 分 → 易大；一攻一守 → 难大 |
| **近 3~6 轮状态** | 连续 5~6 轮大球需提防转小（但平衡性弱于亚盘，反弹概率不如亚盘明显） |

### 3.4 实力悬殊 ≠ 大球（重要纠偏）

- 强对弱不一定大球：弱队太弱进不了、强队心理放松也不猛进，弱队全力收缩反而增强防守。
- 势均力敌也不一定大球：互相克制。
- 打防反的球队不易大球。

### 3.5 大小球赔率围绕让球盘服务（关键认知）

一球盘以上的深盘，大球赔率天然偏低。**不要单纯依据大小球赔率高低判断**，须结合亚盘深浅。

## 四、大小球与亚盘的联动关系（交叉验证核心）

### 4.1 同盘口 + 不同亚盘深度 = 不同解读

同样是高水 2.5 大 / 2.25 大，配不同亚盘深度，结论天差地别：

| 亚盘深度 | 大球盘解读 |
|----------|-----------|
| 浅盘（平/半、半球） | 双方差异小，下盘打出大球正常，高水小盘 + 大 = 正常 |
| 深盘（一球/球半以上） | 上盘很强势，上盘赢盘除 2-0 外都是大球 → **「上盘 + 大」概率很低**，是明显矛盾点 |

### 4.2 浅盘多平局，平局多小球

亚盘浅盘有接近 1/3 比例的平局，平局多小球（尤其 1-1 最多）。
→ 看好浅盘平局 = 看好小球；排除平局 = 自然利大球。

### 4.3 双升水 = 耍诈信号

- 亚盘上盘升水时，大球极难同步降水（理论才有，实战几乎不出现）。
- **亚盘升水 + 大球升水 同时出现 = 双方都在耍诈**：高水方打出可能很小。
  - 若亚盘打出下盘 → 赢球输盘的大球；
  - 若亚盘打出上盘 → 偏偏是 2-0（小球）。

### 4.4 故意错开一级盘（澳彩单盘特性）

澳彩是单盘大小球（无副盘），可故意把盘口错开一级挂着：
- 看好小球：105 水 2.5 大挂着不降盘（哪怕外围已 90 水 2.25 大）；
- 看好 4 球起：0.75 水 2.75 大挂着不开 3.0 大。

这种「故意错开一级」在推演中是重要信号。

## 五、走地（滚球）大小球判断

> ⚠️ 本项目 SOP 以赛前推演为主，走地章节供「即时」节点或用户主动询问走地时参考。
> 📡 系统已支持走地实时播报（/live 订阅），完整走地看盘规则见
> [reference_live.md](reference_live.md)（含封盘/main线/水位突变等实时盘技术信号 + 走地军规 L1~L8）。

### 5.1 总原则

- **走地以大小球为主，让球盘为辅**：让球小于 1 球时任一队先进球都未必真领先（双方进入「赛点」），难判上下盘。
- 追小球需要智慧（一锤定音，错了别盲目补仓）；追大球需要勇气（最后 10 分钟才决定大小，追大别回补小球）。
- 走地两条路：越赢越多 / 越输越多。赢时可跟进但注码不超利润，输时绝不输第三手。

### 5.2 投资有利区域

| 方向 | 底线 | 顶线 |
|------|------|------|
| 小球 | 一般联赛 ≥ 2 球（意乙/法甲等可放松到 1.5/2），低于此放弃 | — |
| 大球 | 小于 1 球坚决放弃 | 最佳 3 球以下（德甲/西甲等可放松到 4 球以下），大于 4 球坚决不追 |

### 5.3 半场 1/1.5 球分水岭

四大联赛及瑞超、实力接近的对决，全场常开 2.5 球、上半常 1 球：
- 上半 0-0 → 中场盘应到 1/1.5 球小球略低水；
- 上半若仍高开 1.5 球 → 庄家看好下半有球，出大球机率相当高。

### 5.4 加减法判断庄家开盘思路

赛前 2.5 球，开赛 10 分钟内进 1 球后：
- 走地仅开 3 球 → 庄家不看好总进球超 3 → **介入小球最佳时机**；
- 走地高开 3.5 球（赛前 2.5+1）→ 不宜介入小球，介入大球需等升水跌盘再出手。

### 5.5 20 分钟法则（走地速查表）

| 进球数 | 大小球盘口 | 进球时间 | 判断 |
|--------|-----------|----------|------|
| 0 | 2 | 无 | 小球 |
| 0 | 1.5/2 | 无 | 放弃 |
| 0 | 2/2.5 ~ 2.5 | 无 | 观望 |
| 0 | 2.5 低水 ~ 2.5/3 | 无 | 分段等高水追大 |
| 1 | 3.5/4 | 前 10 分钟 | 追大 |
| 1 | 3/3.5 ~ 3.5 | 前 10 分钟 | 观望，3 球时开始追 |
| 1 | 2.5/3 ~ 3 | 前 10 分钟 | 追大 |
| 1 | 大 4 | 15 分钟左右 | 观望，小于 4 时追大 |
| 1 | 大 3 | 15 分钟左右 | 观望，小于 3 时追大 |
| 1 | 小 3 | 15 分钟左右 | 追大 |
| 2 | 小 4 | 20 分钟前 | 追大 |
| 2 | 大 4 | 20 分钟前 | 观望，小于 4 追大 |

> 实战需结合亚盘让球与水位。整体联赛同时走地时叠加「每轮大小场次均等」概率：如某轮意甲 8 场，前 20 分钟 5 场有球，则剩 3 场开小机会极大。

### 5.6 走地典型屠杀盘（结合让球盘）

1. 强队让球大 + 进球早 + 进球后让球盘和大小盘下降不大 → 屠杀，亚盘上盘 + 大球同打。
2. 强队让球 + 进球在先，弱队扳平 → 平局表现，追小有收获。
3. 强队让球，弱队进球在先，强队扳平 → 一定还有球进，追大。
4. 弱队领先后收缩 → 强队反超难，**介入小球最佳时机**。

### 5.7 走地小球高水高危

无进球时独赢上盘 + 大球一路升水跌盘。若走地场次小球处于高水赔付方 → 庄家相当看好大球且很快有进球。

## 六、量化校验层（英文 sharp 资料补充）

> 中文盘路偏定性，本章补一个量化锚点，用于交叉验证定性结论。

### 6.1 盈亏平衡锚点

大 2.5 的盈亏平衡赔率约 **2.00**（隐含 ~50% 概率）：
- 大球赔率 < 2.00 → 机构预期高比分；
- 大球赔率 > 2.00 → 机构预期低比分。

### 6.2 去抽水真实概率（no-vig）

判断「价值」前必须先去抽水，不能直接看名义赔率。
**MPTO（按赔率比例分配抽水）公式**：

```text
真实赔率 = (n × 名义赔率) / (n − margin × 名义赔率)
n = 市场结果数（大小球为 2）
```

> 用 Pinnacle / 亚洲盘 / 交易所作为 benchmark，去水后得真实概率，再与自己估算对比找偏差。

### 6.3 over bias（大球偏见）

多数玩家偏好买大，机构（含部分 sharp 书）会把大球赔率人为压短。
→ **职业玩家更多在找「小球」的价值**；强队不保证高比分，别盲跟大球。

### 6.4 进球数据 noisy → 用 xG

单纯用历史进球数预测大小球噪声大（运气成分高）。条件允许时用 xG（预期进球）+ 泊松分布估算各比分概率，比数进球更稳。

### 6.5 Line Freeze（盘口冻结）= 中文「岿然不动」

某方向资金明显涌入但盘口/水位不动 → 庄家认可另一边。
与第 2.3 节「澳门岿然不动 = 真实锚点」、亚盘「阻上型」逻辑一致。

## 七、大小球军规速查表（赛前判断）

> 先用第三章估合理盘口、定开大/开小，再对照下表。重「初盘位置」轻「临场走向」。

**场景：2.25 球初盘（最常见研判区）**

| 编号 | 盘型条件 | 高概率赛果 | 操作 |
|------|----------|-----------|------|

## 八、核心纪律与防错清单

输出大小球结论前逐项自查：

1. **是否以初盘为锚？** 临场走向权重必须低于初盘位置。
2. **是否被「追大」潜意识带偏？** 警惕 over bias，主动审视小球价值。
3. **是否结合了亚盘深度？** 单看大小球赔率高低判断 = 错误（赔率围绕让球盘服务）。
4. **是否区分了真升降水与诱导？** 大小球升降水「假的多」，不变盘才是常态。
5. **2.5 球初盘 + 不动 = 不碰**，不要硬判。
6. **实力悬殊 ≠ 大球**，先看丢球能力（烂盾）与下盘进球能力。
7. **客观因素核对**：联赛球风、天气、赛制、赛程密度、战意。
8. **不死套公式**：红牌/伤病等突发不可预知，结论只能「无限接近」实际值，不可绝对化。

> **置信度上限建议**：纯大小球判断（无亚盘/凯利交叉确认）置信度 ≤ 70；2.5 球模糊盘 ≤ 50。
