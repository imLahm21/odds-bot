# 国家队赛事基本面读法参考手册

> 本文件供 LLM 分析**国家队赛事**时作为**基本面（近况/交锋/赛程/积分榜）专用读法**的规则库参考，
> 体例与 [reference_asian_handicap.md](reference_asian_handicap.md)、[reference_over_under.md](reference_over_under.md) 对齐。
> 触发范围：世界杯、洲际杯（欧洲杯/美洲杯/亚洲杯/非洲杯等）、欧国联、各大洲世预赛、国家队友谊赛等
> **由国家队（而非俱乐部）参赛**的赛事。世界杯只是其中一例，本文件通篇以「国家队赛事」为通用说法。
> 俱乐部赛事（联赛、俱乐部杯赛、欧冠/欧联等）**不适用本文件**，仍按 CLAUDE.md 步骤 1 第 5~7 条的联赛读法执行；
> 俱乐部赛程情境（一周几赛/多线作战/留力）见 [reference_competition_context.md](reference_competition_context.md)。
>
> **定位**：CLAUDE.md 步骤 1（1b 近况 / 1c 交锋 / 1d 赛程积分榜）的联赛读法在国家队赛事下会系统性失真，
> 本文件给出对应的口径切换。盘型/操盘手法层面的国家队赛事教训见
> [reference_case_lessons.md](../实战教训/reference_case_lessons.md) 第十八、十九章，两处配合使用。

## 一、为什么国家队赛事要单独一套基本面读法

联赛读法建立在四个隐含前提上，国家队赛事**逐条不成立**：

| 联赛隐含前提 | 国家队赛事的现实 | 后果 |
|--------------|------------------|------|
| 主队有主场地利 | 赛会制决赛圈**中立场**（东道主除外）；主客场制赛事地利也弱于俱乐部主场 | 主场加成误加，让球深度判偏 |
| 近 N 场同质可比 | 近 N 场混大赛/世预赛/友谊赛，**战意与强度天差地别** | 等权数胜负 → 状态误判 |
| 终指 + 积分榜可锚实力 | 自动流程**无终指**；小组赛积分榜样本仅 1~3 场 | 实力无锚，盘口无参照系 |
| 赛程反映双线/轮换/欧战分心 | 国家队**无俱乐部双线**，真正变量是集训磨合、旅行、体能与停赛 | 赛程维度整段用错 |

因此判定为国家队赛事时，1b/1c/1d 必须整体切换为下列口径。

## 二、先分赛制：赛会制（中立场）vs 主客场制（有地利）

**国家队赛事不是铁板一块，判主客场地利前必须先分赛制**——这是最容易一刀切错的地方：

| 赛制 | 典型赛事 | 场地性质 | 地利读法 |
|------|----------|----------|----------|
| **赛会制决赛圈** | 世界杯正赛、欧洲杯/美洲杯/亚洲杯正赛 | **中立场**（除东道主） | 废除主客场分拆；「主队在左」仅名义，**禁用主场加成** |
| **主客场制** | 世预赛、欧国联、洲际杯预选赛 | **真实主客场** | 有地利，但**弱于俱乐部主场**（无密集主场氛围/裁判倾向相对弱）；高原/长途客场旅行影响可放大 |
| **友谊赛** | 国际比赛日热身 | 名义主客场/中立均有 | 地利**近乎无意义**（练兵为主，见第三章） |

- 赛会制：CSV「主队永远在左」只是名义主队，**不代表有主场**，不得因此上调让球深度或主胜权重。
  仅东道主、或明显球迷地利（如南美球队在南美洲办赛）时给**有限**地利加权，并写明依据。
- 主客场制：主队有地利但打折计，**高原主场（如玻利维亚/厄瓜多尔）、极端长途/气候客场**才显著。
- **无法确定是哪种赛制时，默认按中立场保守处理**（不误加地利），并在报告中标注不确定。

## 三、近 N 场必须按赛事性质分层，禁止等权数胜负（对应 1b）

国家队近 N 场常混三类，参考价值差异极大，**先给每场标 (a)/(b)/(c) 再加权**：

| 类别 | 场景 | 战力参考权重 | 进失球参考 |
|------|------|--------------|------------|
| (a) 正式大赛 / 世预赛关键战 / 欧国联 | 出线/淘汰赛/强强对话 | **高**（真实战力） | 真实 |
| (b) 世预赛虐鱼腩 / 出线无忧走过场 | 强队打弱旅、大比分刷球 | 低（胜负注水） | **进球数注水，须打折** |
| (c) 热身友谊赛 | 练兵、轮换、试阵 | **极低**（近乎无参考） | 只看结构不看比分 |

- **友谊赛连胜 ≠ 状态好，友谊赛连败 ≠ 状态崩**。友谊赛胜负近乎不计入状态趋势。
- (b) 类大胜的进球数不可直接推「进攻火爆 → 利大球」，要看对手档次。
- 只有 (a) 类的连续表现才构成可信的「状态趋势」。

## 四、实力锚替代——无终指时用 FIFA 排名 / 洲际强弱 / 世预赛含金量（对应 1b/1d）

联赛靠终指 + 积分榜锚实力；国家队自动流程无终指、决赛圈小组赛积分榜样本仅 1~3 场，**不足以锚实力**。改用：

1. **世界排名档次差**：按 FIFA 排名/公认强弱分档（顶级 / 一流 / 二流 / 鱼腩），看两队跨几档。
2. **洲际强弱梯度**：欧洲/南美 > 非洲/亚洲 > 中北美/大洋洲为**传统**梯度，
   但**必须按当届状态修正**（黑马、换代、临阵伤病），不可只按洲际刻板印象。
3. **世预赛出线含金量**：欧洲区出线难度 > 亚洲区直接出线；从强区厮杀出线的球队实力锚更高。

> **用法**：把上述实力锚与盘口方向对照——**盘口与实力锚一致才可提置信度**；
> 盘口方向与实力锚背离时，优先怀疑诱盘（见 [reference_case_lessons.md](../实战教训/reference_case_lessons.md) 第十八章），不可提置信度。

## 五、赛程改判集训 / 体能 / 停赛，废除「双线 / 分心 / 轮换」（对应 1d）

国家队无俱乐部双线作战、无欧战分心。1d「未来 5 场赛程密度 / 双线 / 临近强敌」在国家队赛事下**整段改判**：

- **集训磨合**：国际比赛日临时集结，磨合时间短；大赛前长期集训的队默契更好。
- **休息天数 / 密度**：决赛圈**3~4 天一场**为高强度，体能是深盘兑现的硬约束。
- **加时消耗**：淘汰赛上一轮是否打满加时/点球？打过加时的一方体能透支，让球兑现力下调。
- **核心停赛 / 伤病**：累计黄牌停赛、红牌、主力伤病——国家队阵容薄，核心缺阵杀伤远大于豪门联赛。
- **旅行 / 时差 / 气候**：主客场制赛事中，长途客场、跨时区、高原/高温客场须下调客队战力。
- **是否已提前出线**：小组末轮已出线的队可能轮换主力（→ 战力下调、利小球/冷平，见第六章）。

> 深盘让球方若**刚打过加时**或**核心停赛**，须下调让球兑现力，并在风险提示中标注。

## 六、小组赛出线形势与战意——冷平/爆冷是结构性尾部事件（对应 1d + 战意研判）

自动流程的积分榜已抓小组表 + 最佳第三名（见 `bot/fundamentals.py`），必须**显式研判出线形势**，不能只列排名：

- **净胜球 / 末轮出线数学**：算清每队「打平即出线 / 必须赢 / 赢几球才出线」的确切格局。
- **已出线 → 留力轮换**：已锁定出线的队常轮换主力、降低强度 → **利小球、利冷平**。
- **生死战 → 必须赢**：必须取胜的一方强攻 → 利大球；但压力下也可能**心态崩盘**，两种尾部都要保留。
- **默契球风险**：双方**各取一分即双双出线**时，易踢出小比分/闷平 → **大幅利小球与冷平**，末轮尤其警惕。

**战意与心理（贯穿小组赛与淘汰赛）**：

- 小组赛强队已出线**松懈**、末轮**留力**；淘汰赛**保守求点球**（利小球 + 平局/冷平）；弱队**龟缩摆大巴**。
- **强弱深盘 + 高平赔时，冷平 / 受让方直接取胜必须保留独立高权重**——
  这是结构性尾部事件，不是噪音（详见 [reference_case_lessons.md](../实战教训/reference_case_lessons.md) 第十八、十九章）。
- **不得因「实力悬殊」把平局 / 爆冷概率归零**。平赔越高，一旦冷平兑现破坏性越大。

## 七、H2H 权重比联赛更低（对应 1c）

- 国家队交锋常**跨届、跨完整换代阵容**（间隔数年、球员全换），历史交锋与当下战力关联极弱。
- **友谊赛交锋 ≠ 大赛交锋**，不可等同看待。
- H2H 仅作**极弱背景参考**：**本届/本阶段状态 + 实力锚（第四章）权重 ≫ 历史交锋**。
- 老交锋的比分规律**不作大小球依据**（阵容战术早已不同）。

## 八、国家队赛事基本面防错清单（研判国家队赛事时逐项执行）

- [ ] 是否为国家队赛事（世界杯/洲际杯/欧国联/世预赛/友谊赛）？若是，是否已切到本文件口径，而非联赛读法？
- [ ] **是否已先分赛制？** 赛会制决赛圈=中立场废除地利；主客场制=有地利但打折；友谊赛=地利近乎无意义（不可一刀切中立场）。
- [ ] 近 N 场是否已按 (a) 大赛 / (b) 世预赛虐菜 / (c) 友谊赛**分层加权**，而非等权数胜负？
- [ ] 友谊赛连胜/连败是否已**排除**出「状态趋势」结论？(b) 类大胜进球数是否已按对手档次打折？
- [ ] 无终指时，是否已用 **FIFA 排名 / 洲际强弱 / 世预赛含金量**建立实力锚，而非无参照系裸推？
- [ ] 盘口方向是否与实力锚一致？**背离时是否怀疑诱盘、未提置信度**？
- [ ] 赛程是否已改判**集训磨合 / 休息天数 / 加时消耗 / 核心停赛 / 旅行气候**，而非套用「双线/分心/轮换」？
- [ ] 深盘让球方是否刚打过加时或核心停赛？若是，是否已**下调让球兑现力**？
- [ ] 小组赛是否已**算清末轮出线数学**（打平即出线/必须赢/默契球）？
- [ ] 是否存在**已出线留力**或**默契球**格局？若是，是否已上调**小球 + 冷平**权重？
- [ ] 强弱深盘 + 高平赔时，是否为**冷平 / 受让方直接取胜**保留了独立高权重（承接第十八、十九章）？
- [ ] H2H 是否只作**极弱背景参考**（跨届换代/友谊赛不可等同大赛），未用老交锋比分规律定大小球？
